# 军事视频 Prompt 模板体系 v3.0

> 创建日期：2026-07-18
> 最后更新：2026-07-18（v3.0）
> 核心改造：从「空镜美学」到「文章锚定叙事」
> v3.0 新增：R6(物理合理规则) + R7(肖像合规规则) + 无文字约束 + 场景光线改为正午明亮日光
> 适用场景：今日头条军事号 — 文章→CG 视频分镜生成

---

## 一、核心哲学：从"空镜美学"到"文章锚定"

### v1.0 问题诊断

v1.0 的视频 Prompt 遵循了 Seedance 2.0 导演引擎的"一个意图统领所有乐器"法则，但错误地将"意图"理解为**氛围情绪**（紧张/震撼/压迫感），而非**文章叙事**。

v1.0 的典型 Prompt 结构：
```
[CG风格前缀] + [场景氛围描述] + [光影] + [镜头运动] + [约束]
```
缺失了最关键的一环：**这个镜头在讲文章中的哪句话？**

结果是 4 段视频全部是高质量的空镜——视觉上震撼，信息量上为零。观众看完视频不知道文章说了什么，因为视频根本没有进入文章的内容层。

### v2.0 核心原则

> **视频是文章的视觉翻译，不是文章的"氛围配乐"。**

- 每个镜头必须可以追溯到文章中的**一个具体句子**或**一条具体逻辑链**
- Prompt 中必须有文章中出现过的**具体实体**（航母名称、装备型号、基地名称、数据）
- 视频必须有**信息传达功能**——不是"让观众感觉到什么"，而是"让观众看到文章说了什么"

### 三类视频功能

| 类型 | 功能 | 在 Prompt 中的体现 |
|------|------|-------------------|
| **展示型镜头** | 把文章中描述的场景/装备/事件可视化 | 直接出现文章实体：林肯号航母、第五舰队指挥中心被毁、加油机损失 |
| **对比型镜头** | 展示文章中的对比/消长关系 | 两艘航母→只剩一艘 / 弹药充足→弹药架空荡 / 油价攀升数据曲线 |
| **隐喻型镜头** | 把文章的抽象结论转化为具象画面 | "战场上得不到的"→空谈判桌，但必须有前文铺垫，不能凭空出现 |

## 二、实体锚定规则（R1）

### 规则

> **每个 segment 的 Prompt 必须包含至少 3 个文章中出现过的具体实体。**

"具体实体"定义：文章中明确提及的、有具体名称/型号/数据的事物。不包括泛化词（"基地""战机""爆炸"）。

### 实体清单提取法

在写 Prompt 之前，先从文章中提取实体清单：

```
文章路径 → 逐段扫描 → 提取：人物/装备/基地/组织/数据/事件 → 形成「段-实体」映射表
```

#### 示例：美军再打伊朗能打多久 — 实体清单

| 文章段落 | 提取的实体 |
|---------|-----------|
| 第1-2段 | 特朗普、「彻底摧毁伊朗在霍尔木兹海峡所有军事能力」、美军中央司令部、连续第三个晚上 |
| 第3-5段 | **林肯号航母**、两艘航母→只剩一艘、300+作战飞机、超6个月部署、**加油机**、**预警机**、远程奔袭 |
| 第6段 | 精确制导武器、生产周期数月、弹药告急、欧洲/亚太战区调拨 |
| 第7-8段 | 伊朗导弹、**伊朗无人机**、美军中东基地、雷达被毁、**巴林第五舰队作战指挥中心**被打没 |
| 第9段 | 科威特、巴林、海湾国家、单独和平协议 |
| 第10段 | 国会、议案限制动武权限、特朗普与国会争执、阿富汗伊拉克覆辙 |
| 第11段 | **国际油价87美元**、突破100美元、中期选举、通胀 |
| 第12-13段 | 「还有可能与伊朗达成协议」、停火谈判 |

### 反例与正例

| | v1.0（空镜） | v2.0（锚定） |
|---|---|---|
| **01段** | "中东沙漠黄昏，残阳在地平线投下最后一抹橙金余晖。前景左侧沙丘起伏，右侧地平线上一座军事基地多处建筑同时燃烧" | "波斯湾黄昏，一艘大型航母——「林肯」号——孤零零停在海上，甲板上仅停着稀疏几架战机。远处伊朗方向，导弹尾焰划破天空。" |
| **02段** | "黄昏残阳斜射进美军基地，在地面拉出长长金属阴影。连续横移长镜头缓慢扫过三个空间" | "加油机停在跑道尽头、输油管耷拉在地；预警机雷达盘被击穿一个大洞；弹药库混凝土掩体一排排空架——镜头横移扫过三种「打不起」的证据。" |

## 三、关键字注入规则（R2）

### 规则

> **Prompt 中必须注入文章的核心关键字。这些关键字是观众读文章时留下的记忆锚点，视频必须呼应它们。**

### 关键字注入方式

| 注入方式 | 说明 | Prompt 示例 |
|---------|------|------------|
| **直接命名** | 用文章原词描述场景主体 | "林肯号航母的舰岛在夕照中拉出长长阴影" |
| **数据可视化** | 将文章中的数据转化为画面元素 | "油价表从87美元跳向100美元，数字变成红色" |
| **事件再现** | 将文章描述的事件转化为可见动作 | "伊朗无人机撞向巴林基地的指挥中心大楼" |
| **对比呈现** | 将文章的对比关系转化为构图对比 | "画面左侧：2月两艘航母并排；画面右侧：现在只剩林肯号孤舰" |

### 关键字注入强度表

| 等级 | 要求 | 适用场景 |
|------|------|---------|
| **L1 轻度** | 每段 2-3 个关键字 | 地缘政治/宏观分析类文章 |
| **L2 中度** | 每段 3-5 个关键字 | 装备分析/战力对比类文章 |
| **L3 重度** | 每段 5+ 个关键字 | 战况报道/事件实录类文章 |

## 四、逻辑链强制规则（R3）

### 规则

> **每段视频必须对应文章的一段逻辑链。Prompt 需展现该逻辑链的因果过程，而非仅呈现结果状态。**

### 逻辑链映射法

从文章中提取逻辑链：

```
文章句子："长期部署的航母，战斗力随舰载机维护、人员疲劳、设备检修而断崖式下降。"
         ↓
逻辑链: [超期部署] → [维护不足] → [人员疲劳] → [战斗力断崖下降]
         ↓
Prompt可视化: "林肯号航母甲板上仅剩零星几架战机，机库内维修人员精疲力竭地检修引擎，
              一架战机翼尖冒出火花——维护赶不上损耗。"
```

### 逻辑链 vs 氛围对比

| 文章原文 | v1.0（氛围） | v2.0（逻辑链） |
|---------|-------------|---------------|
| "弹药库存同样告急，精确制导武器生产周期长达数月" | "弹药储存区混凝土掩体一排排空荡" | "精确制导炸弹的生产线——原材料区标注'到货日期：3个月后'，组装台上的零件散落，成品架上挂'库存0'标签。生产周期数月，空袭每打一发就少一发。" |
| "伊朗的反击，直接把美军在巴林的第五舰队作战指挥中心打没了" | "碎裂屏幕墙闪烁红色故障信号光" | "伊朗无人机从画面右侧飞入，撞向巴林基地标有'第五舰队'的建筑。爆炸瞬间，建筑从中间塌陷，通信天线折断，屏幕墙全部变红闪烁'CONNECTION LOST'。" |

## 五、主角在场规则（R4）

### 规则

> **视频中必须有可识别的主角元素。观众能一眼认出'这是美军'或'这是伊朗'或'这是文章中的那个东西'。**

### 主角类型与 Prompt 策略

| 主角类型 | 视觉锚定方式 | Prompt 策略 |
|---------|------------|------------|
| **国家/阵营** | 装备特征+涂装+标志性轮廓 | 美式航母舰岛靠右/S形斜角甲板；俄式圆顶雷达罩；伊朗沙色涂装无人机 |
| **装备** | 型号特征+上下文环境 | 林肯号=尼米兹级平顶航母，斜角甲板，舰岛靠右 |
| **地点** | 地理标志+地图标注 | 巴林基地=波斯湾岛屿+美军标识；霍尔木兹海峡=狭窄水道+油轮 |
| **人物** | 剪影+标志性动作/环境 | 特朗普=讲台剪影+发言手势（非面部特写，CG 风格合规） |
| **数据** | 数字+趋势曲线+颜色标注 | 油价87→100 红色上升箭头；弹药库存：2月3000→7月200 |

### 主角缺席 vs 主角在场

| | v1.0 | v2.0 |
|---|---|---|
| **01段** | 无名基地+无名战斗机 | **林肯号航母**（舰岛靠右+斜角甲板）+ **特朗普剪影**（讲台发言） |
| **02段** | 无名运输机+空弹药库 | **KC-135加油机**（输油管特征）+ **E-3预警机**（雷达盘特征）+ 精确制导炸弹生产线 |
| **03段** | 无名指挥中心 | **巴林第五舰队指挥中心**（海湾岛屿+建筑标牌）+ **伊朗Shahed无人机** |
| **04段** | 空谈判桌 | 特朗普白宫剪影 + 油价87红色箭头 + 中期选举投票图 + 空谈判桌 |

## 六、物品规则（R5）

### 规则

> **视频中必须有具体可辨识的军事物品。不是"一架飞机"，而是"一架后掠翼双垂尾战斗机，机腹挂载两枚精确制导炸弹"。**

### 军事物品分级（从抽象到具体）

| 等级 | 描述 | Prompt 示例 |
|------|------|------------|
| **L0（禁止）** | 纯抽象，无任何具体物品 | ❌ "军事基地在燃烧" |
| **L1（最低）** | 至少一个具体装备类型 | "后掠翼双垂尾战斗机" |
| **L2（标准）** | 具体装备类型+可辨识特征 | "后掠翼双垂尾战斗机，机翼下挂载两枚精确制导弹药" |
| **L3（优）** | 具体型号特征+上下文+状态 | "一架F-15风格的双垂尾后掠翼战斗机停在林肯号甲板上，座舱盖打开，维修梯搭在机身侧面——行动中的维护" |

### 军事物品词库（常用装备的 Prompt 描述）

| 文章关键词 | Prompt 描述（CG风格，不写品牌名） |
|-----------|-------------------------------|
| 航母 | 大型平顶航母，斜角甲板，舰岛靠右，甲板上停着数架折叠翼舰载机 |
| 加油机 | 大型四发加油机，翼尖有加油吊舱，输油管从机尾垂下 |
| 预警机 | 大型预警机，机背装有圆盘形雷达天线罩，四发涡扇引擎 |
| 精确制导弹药 | 圆柱形弹体，前部有激光导引头圆窗，后部四片梯形稳定翼 |
| 伊朗无人机 | 三角翼无人机，沙色涂装，机头尖锐，翼尖上翘 |
| 弹道导弹 | 圆柱形多级弹体，圆锥形弹头，尾部四个梯形稳定翼，从沙漠发射井升起 |
| 指挥中心 | 低矮混凝土建筑群，屋顶布满天线和球形卫星通信罩 |

## 六-B、物理合理规则（R6）——2026-07-18 新增

### 规则
> **视频中所有物体的空间位置必须物理合理。飞机在空中飞，船在水上浮，天线在建筑屋顶，人和物品不超出其自然活动框架。**

### 常见物理错误（禁止例）
| 错误类型 | 说明 | 禁止 |
|---------|------|------|
| **位置错位** | 飞机飞在水平面而非空中 | ❌ "飞机贴着水面滑行" → ✅ "飞机在500米高度巡航" |
| **物体畸形** | 舰船出现不属于该类舰船的部位（桅杆方向反、甲板结构异常） | ❌ "舰岛出现在航母左舷" → ✅ "舰岛靠右的平顶航母" |
| **人物越界** | 人物超出其自然活动框架（从讲台伸出、离开场景） | ❌ "特朗普从讲台走向窗前" → ✅ "特朗普剪影在讲台后方保持演讲姿态" |

### R6 强制检查清单
| # | 检测项 | 通过标准 |
|---|--------|---------|
| R6-1 | 位置合理 | 所有移动物体在正确介质中飞行/航行/行驶 |
| R6-2 | 物体完整 | 所有物体形态符合该类物体的标准特征，无额外/缺失部件 |
| R6-3 | 人物框架 | 人物剪影不超出其自然活动范围（讲台/椅子/窗边） |

## 六-C、肖像合规规则（R7）——2026-07-18 新增

### 规则
> **任何人物的视觉呈现必须避免侵犯肖像权。采用「神似剪影」策略——轮廓相似但无真实面部特征，不表现可识别的面部细节。**

### 肖像合规策略
| 方式 | 说明 | Prompt 示例 |
|------|------|------------|
| **剪影法** | 人物用全黑/半透明剪影，无面部细节 | "剪影轮廓站在讲台后方，背光照亮肩线轮廓" |
| **背光法** | 人物背对镜头，面部不可见 | "背对镜头的领导人剪影站在窗前" |
| **术语模糊法** | 用职衔替代姓名，降低模型对特定人物的绑定 | ❌ "特朗普" → ✅ "美军最高指挥官剪影" 或 "白宫领导人剪影" |
| **极简轮廓** | 仅用肩线+发型轮廓暗示身份 | "肩部以上轮廓，后梳发型剪影，无任何面部细节" |

### R7 强制规则
| # | 规则 | 说明 |
|---|------|------|
| R7-1 | 无真实人脸 | 禁止任何人物的面部特征（眼睛/鼻子/嘴巴/皱纹），仅用剪影/轮廓 |
| R7-2 | 无动作越界 | 人物剪影保持静态/微动态，不超出其自然环境框架（讲台后/窗前/桌前） |
| R7-3 | 职衔优先 | 优先用「指挥官剪影」「领导人剪影」「白宫决策者」替代具体名字 |

## 七、Prompt 七槽公式（军事文章锚定版 v3.0）

在原有 Seedance 2.0 Director Formula 基础上，增加「文章锚点」为首槽：

```
[文章锚点] + [主体锚点] + [可见节拍/动作] + [场景/环境] + [单一动机运镜] + [光影/风格] + [约束]
```

| 槽位 | 说明 | 军事场景填充 | 强制要求 |
|------|------|------------|---------|
| **① 文章锚点** | Prompt 中植入的、与文章原文对应的实体/关键字/数据/逻辑 | "林肯号航母""第五舰队""精确制导弹药""油价87美元""超期部署6个月" | **每段≥3个** |
| **② 主体** | 模型必须追踪的视觉锚点 | "林肯号航母舰岛" | 必须是文章锚点中的具体实体 |
| **③ 动作** | 唯一的可见变化（非静态） | "伊朗无人机撞向巴林基地大楼" | 必须有因果动作，不能只是"燃烧" |
| **④ 场景** | 不可见的环境补充 | "正午波斯湾晴空，巴林岛屿的军事基地" | 含地理位置/时间锚定；场景光线统一为正午明亮日光 |
| **⑤ 相机** | 一个主要运动及终点 | "镜头从航母甲板推入林肯号舰岛" | 服务于信息揭示，非装饰性 |
| **⑥ 光影/风格** | 物理光源+风格描述 | "正午强烈顶光，高对比明暗，晴空万里，一切物体清晰可见" | 光源必须有世界内来源；**禁止黄昏/逆光/暗调** |
| **⑦ 约束** | 保留项与排除项 | "无任何文字/数字/标签/字幕；物体物理位置合理（飞机在空中/船在水面/天线在楼顶）；人物剪影仅限静态轮廓、无面部特征、不超出任务框架；所有事物可辨识；CG数字绘风格" | 含军事合规约束；**禁止任何文字元素；禁止肖像特写；禁止物理错位** |

## 八、验收清单（Prompt 级）

每段视频 Prompt 写完必须过以下 6 项：

### 文章锚定检测（新增）

| # | 检测项 | 通过标准 |
|---|--------|---------|
| A1 | 实体锚定 | Prompt 中包含 ≥3 个文章具体实体（非泛化词） |
| A2 | 关键字注入 | Prompt 中出现 ≥3 个文章原文关键字/短语 |
| A3 | 逻辑链对应 | Prompt 展现的行动与文章某条逻辑链的因果过程匹配 |
| A4 | 主角在场 | 观众能识别出至少一个明确主角（阵营/装备/地点/人物/数据） |
| A5 | 物品可辨识 | 至少有一个军事物品有具体特征描述（L2级或以上） |

### 一致性出厂测试（原有，保留）

| # | 检测项 | 通过标准 |
|---|--------|---------|
| C1 | 一句话意图清晰 | 能一句话说清"这段视频在讲文章中的哪件事" |
| C2 | 乐器一致性 | 景别/运动/光影/表演都服务于同一意图 |
| C3 | 表演可见性 | 动作是可见物理变化，非情绪标签 |
| C4 | 光源动机 | 光源有世界内来源，摄影机运动有叙事理由 |
| C5 | 风格合规 | 符合 CG 数字绘 + 军事合规（无真实人脸/国旗/文字） |
| C5a | 无文字 | 画面中无任何可识别的文字/数字/标签/字幕——所有信息通过画面本身传达 |
| C5b | 肖像合规 | 人物呈现使用剪影/背光/轮廓，无面部特写，不侵犯肖像权 |
| C5c | 物理合理 | 所有物体在正确位置（飞机空中、船在水面、天线在楼顶），无不合理形态 |
| C6 | 复杂度可控 | 关键元素（装备形状/爆炸/数据标注）在模型能力范围内 |

### 验收门禁

- **A1-A5 全部 PASS** → 进入一致性测试
- **A1-A5 有 FAIL** → 回到读解阶段重写
- **C1-C6 全部 PASS** → 交付
- **C1-C6 有 FAIL** → 回到对应乐器调整（≤2次）
