晋，读音为jìn，是汉语一级通用规范汉常用字。晋的主要字义，一是进或升。如晋级、晋见、晋升等。二是用为国名、地名、姓名等。

“晋”字初文始见于商代甲骨文。目前发现的甲骨文中使用 "晋" 字的实例有限，《甲骨文合集》19568（宾组卜辞） 仅存 "晋扶" 二字，上下文缺失，无法确定完整含义。《甲骨文合集》27710："戊申卜，贞：王其田于晋，湄日亡灾？"《前》6・58・4："贞乎（师）般伐晋（方）"等多为方国名或地名。

关于“晋”的字形和本义有多种解释。

一是认为“晋”是一个会意字，本义是进。《说文解字》：“晋，进也。日出万物进。从日从臸。《易》曰：“明出地上，晋。”徐铉注曰：“臸，到也。会意”段玉裁也认同许慎”晋的本义是进“的观点，并对其会意逻辑作出进一步说明。认为不仅日出万物进，而且人们也是日出而作，由这样的自然现象会意并引申出“进”的含义。《说文解字注》：“日出而萬物進。故其字从日。从日从臸。臸者，到也，以日出而作會意。”“晉，進也。以曡韵爲訓。”但段注又言：“凡進皆曰晉，難進亦曰晉。”这就让人有点自相矛盾的感觉。怎么可能如段所言：进是晋，难进也是晋呢？

二是认为“晋”是一个象形兼会意字。字形象“二矢射中靶心”，本义是进、升。认为甲骨文“晋”上部像两支倒立的箭（“矢”），下部是像“日”字的靶心，这就是箭要射中的目标。整个字形表示弓箭一支接一支地射中了靶心。金文字形与甲骨文大致相同。小篆的“晋”字，上面变为两个“至”。甲骨文的“至”指落到地上的箭。其义与甲骨文的“晋”一样，指两支箭射中靶心射箭者连续射中目标，就有进一步、跟进、递进的意思。所以“晋”的本义就是进、升的意思，如：晋升、晋见、晋级。隶书和楷书将上部的两支箭大大简化，已看不出箭形。

著名文字学家唐兰、李学勤、裘锡圭、于省吾、徐中舒等均持如此相同或相似的观点。并使之成为主流观点。唐兰《古文字学导论》认为："晋从二矢从日，日即靶的象形，二矢射中，会意为进。"李学勤《字源》指出："晋字甲骨文字形上部如两支倒立的箭，下部如靶心，整个字形表示箭矢连续射中目标。" 裘锡圭《文字学概要》中认为："甲骨文 ' 晋' 字从二矢从日，日代表箭靶，两矢并列表示连续射中，本义为 ' 进'"。于省吾《甲骨文字释林》提到晋字 "像两矢射日（靶）之形，日为靶的象征"。徐中舒《甲骨文字典》认为晋字 "从二矢从日，日像靶心，会二矢射中之意"。

三是同样认为“晋”字是一个象形兼会意字，本义是进、升。同样认为字中的上部为两支箭矢，但字中的下面不是箭靶，就是“日”（太阳）。其所描绘的是：两支箭一起射向那空中的太阳，即学者们所说的“二矢射日”。据此，有的学者甚至认为“晋”字是先民根据“后羿射日”的传说创制的，它的本义指射日之箭。由于“晋”的初形为“二矢射日”，其中包含有前进、上升之义，因此，“晋”便引申为“进、升”等意义。

四是同样认为晋是一个象形兼会意字，是搢的本字，本义插。杨树达在《积微居小学金石论丛・释晋》中明确提出："晋字上像二矢，下为插矢之器"，"二矢插器，其义为箭" • 他认为下部的 "日" 不是靶心，而是盛放箭矢的容器，整个字形表示 "两支箭插在箭壶中"。晋的古字形像两支箭放在匣、函中，本义指插，是“搢”的初文。这个意思后来写作“搢”。箭射出去有向前向上的趋势，所以“晋”表示前进和上升。

许慎《说文》及各注说“晋”从日从臸，臸的含义是至、到。其表义逻辑有点牵强附会。且对至的解释没有充分考虑甲骨文等古文字中至、臸的字形及字源本义及其差别。现代文字 学家的各说，考虑了晋的甲骨文等古文字特点，但都把上面的“臸”解读成两只箭矢、但实际上其与箭矢相差甚远，不仅类似箭头的方向不对，其形与箭头也不一样。上面的部件，实际上是两个颠倒的“交”。至于下面的“日”无论从甲骨文到金文以及楚帛和小篆字形均为“日”（太阳），与箭靶没有关系。而且，没有出土文物表明，在“晋”最初 字创制的商代之前，有象后世那样的转型箭靶。而在古文字中，用于表示容器或函的符号中“口”不是“日”。

晋字的甲骨文等古文字上面的部件是两个“交”字的颠倒而不是矢。关于这一点，我们可以比较晋、矢、至、臸、交的古文字字形，从而不难得出结论。“晋”的读音可能也与其字中的“交”有关。

交和矢在甲骨文中字形极为相近，表明其原本为一个字。均为一个正面的人，双腿相交叉或交替的意象，其主要来源于人行走时两腿交替向前的直接描绘。《说文》：“交，交脛也。从大，象交形。”“引申之爲凡交之偁。”“凡㒳者相合曰交，皆此義之引申叚借耳。”

许慎《说文》在解释矢时言：“弓弩矢也。从入，象鏑栝羽之形。古者夷牟初作矢。”许慎认为“矢”的本义是古代的箭的象形。但是，我们看矢的甲骨文、金文字形也根本不象箭的形状。同时，“矢”的字义不仅可以指箭，同时又陈列、陈述、施布、直、正直等义，并可通誓、施。如果，矢的本意就是箭矢，很难完全合逻辑地引申出上述全部字义。但如果我们把“矢”与“交”看作同源，则上述问题可以迎刃而解。“交”来源于人迈步行走即行走时双腿交替向前的动作，这与箭离开弓向前射出是相同的，因此，在甲骨文中，以“交”和“矢”两字基本相同。由于文字的分工分化，才分别有了“交”和“矢”两个字，尤其是金文中交和矢的分化分工更加明显、明确。

至，许慎在《说文》中解释：“至，鳥飛从高下至地也。从一，一猶地也。象形。不，上去；而至，下來也。”许奶粉和人解释完全是附会“至”小篆字形。至与鸟飞上去飞下来、飞不上去飞不下来，完全没有关系。至的甲骨文和金文字形，至字上面是一个倒立的“交”，正面是一横或一竖折。就是一个人走到某一确定的地点的意象。因此，至引申为到、来，又引申为最。至的字形与人双腿交替走路有关，不仅体现在至的含义中，楚帛“至”字中的交讹变得很象“止”或“之”，也是明证。

臸，《说文》：“臸，到也。从二至。”段玉裁《说文解字注》：“(臸)到也。不言至言到者、到者至之得地者也。”“��、近也。从聲。然則二至當重不當竝。”为何臸当作重，而不当作并讲呢？个人认为这是因为至中的两个“至”一个是指行走到某一地点的人，一个是指此人的影子，故曰重。不是两个人一块到某一地点，而是人及其影子同时到了某一地点。这就是段所言“不言至言到者、到者至之得地者也”。

弄清楚了交、矢、至、臸，对于“晋”的字形及本义的理解就容易多了。

甲骨文等古文字中的“晋”，上面是臸，正面是日，其所描述的意象就是，在太阳光下，一个人向前一步，其影子跟着向前一步。孩童之时，我们常常会观察阳光下影子随人前行的情景，并由此产生好奇。所以，古人就是根据自身的这种最常见、最普通的观察创造了“晋”字。因此，“晋”的字源本意就是人向前走一步，在阳光照射的土地上影子移动的情景，由此会意出进的含义，引申出升的含义，还引申出光明、亮丽之义。因此，晋的字义中包含着从原点到终点的内涵。晋级、晋升、晋见，都有前进一步的含义。

晋也是搢、缙等字的声旁。我们已经知道，绝大多数形声字的声旁不仅表音，同时表义。正因为晋有进的含义，所以在前面加上“手”就是搢，其的字义为插或插进。瑨，把晋的字源本义代入到瑨中，瑨就是指似玉的美石。代入到溍中，溍就是指奔腾向前的水流。如 “川流溍溍，汇于东海”。即指水势缙，《周礼・春官・典瑞》："王晋大圭"，郑玄注："晋读为搢绅之搢，谓插于绅带之间。""缙绅" 即 "搢绅"，意为" 插笏于绅 "(将笏板插入腰带)。"缙" 的造字本义：官员上朝时用来插笏的丝织衣带。因为这种丝织衣带古时一般为赤色，所以《说文》："缙，帛赤色也。"由于"晋" 又有 "光明、鲜艳" 的引申义关联，所以缙引申指赤色丝帛。又因"晋" 的基本义是 "进、上升"，故" 缙 " 字又指做做过官的人，"缙绅" 代指官僚士大夫阶层。

从甲骨卜辞的记载来看，商代即有晋方即商代晋国。西周时期，周成王灭此地的祁姓唐国即商代即存在的唐国，将其地改封给叔虞，建立姬姓唐国。而祁姓唐国的人，则被迁到位于今陕西的镐京附近，建立了杜国。叔虞因此称为唐叔虞。《史记》等史书记载，唐叔虞去世后，其子燮父继位，在位期间，迁都晋水之滨，因此将唐国改称晋国。这就是史书记载的西周和春秋时期晋国的由来。但是，我们还可以从史书的记载看到，唐叔虞的后代还在南阳盆地建立有唐国。这是被迫的还是其它的原因。换句话说，晋侯燮父改唐为晋的历史真相到底是什么？这与西周的分封制度有什么关系，都是值得探讨的。
