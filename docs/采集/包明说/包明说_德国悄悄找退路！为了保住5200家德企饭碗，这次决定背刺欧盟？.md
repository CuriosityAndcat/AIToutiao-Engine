近日，中国商务部部长王文涛会见了德国联邦经济和能源部部长赖歇。两人谈完之后宣布了一件事：中德经济合作联委会机制正式重启。

双方要在联委会下面设两个工作组，一个管贸易投资政策沟通，一个管产业合作政企对话。出口管制也要马上启动技术层面磋商。

中德经济合作联委会，上一次会议是2016年11月在北京召开的。此后近十年间，这一正部级机制因中欧地缘政治摩擦加剧而中断。

一个停了十年的机制突然复活，为何选在这个时间点重启？

其实，这是德国在中欧贸易战的大背景下，给自己单独开了一条“逃生通道”。

6月3日，欧盟27国投票通过对中国电动汽车加征最高35.3%的反补贴税。紧接着欧盟又要对中国插电混动车下手，政策准备已经就绪，就等成员国批准。

欧盟还在酝酿压缩全球钢铁免税进口配额，中国作为欧盟第一大钢铁进口来源国首当其冲。法国总统马克龙高调倡议“欧洲版301条款”，G7峰会把中国商品出口列为核心议题。

整个欧盟都在往对华强硬的方向狂奔。但德国呢？德国《法兰克福汇报》的评论说得明明白白：“对华加征关税救不了欧洲工业”。

德国《商报》也承认，德国汽车、化工、机械等支柱产业高度依赖中国市场，反制风险让德国企业不愿冒险。

为什么德国这么怕？因为数据摆在那里。2025年德国对华新增投资约70亿欧元，同比暴增55%以上，创下2021年以来的最高水平。中德贸易额达到2530亿欧元，中国再次成为德国最大的贸易伙伴。

目前在华德企超过5200家，中国德国商会的调查显示，93%的受访德国企业计划继续留在中国，53%计划增加在华投资。

2530亿欧元的生意，5200家企业的饭碗，你说断就断？德国政府要是跟着欧盟的节奏走，第一个被国内企业骂死的就是他们自己。

德国在欧盟对华政策上一直扮演“刹车片”的角色。法国要加税，德国说不行；欧盟要搞“欧洲版301条款”，德国说冷静，但德国的反对声越来越不管用了。

欧盟的决策机制是多数表决，法国带着一帮成员国往前冲，德国一个人拉不住了，所以德国换了一个打法。欧盟层面我拦不住，那我就在双边层面自己解决。

中欧刚坐下来谈，德国就拉着中国单独开小灶。你不是要打贸易战吗？我先把双边对话管道修好。

联委会下设的两个工作组也很有讲究。贸易投资工作组管政策沟通，产业合作工作组管政企对话。一个对上，一个对下。对上跟中国政府谈政策，对下跟中国企业谈合作。两条腿走路，把中德经贸关系死死绑住。

王文涛在会上强调中欧经贸本质是互利共赢，呼吁德方在欧盟内发挥建设性作用。赖歇也表态，愿深化人工智能、供应链等领域合作，共建互信可持续伙伴关系。

这话翻译过来就是：你欧盟要打是你的事，我德国跟中国的生意照做。

德国政府的转向不是凭空来的，是被企业逼的。德国经济研究所的数据显示，2025年德国对华投资超过70亿欧元。德国联邦外贸与投资署的数据也印证了这一点。

5200家在华德企，没有一家希望中欧贸易战打起来。德国《明星》周刊的报道说得很直白：许多德国企业在中国投资建厂，并将中国视为关键出口市场，企业界担心欧盟的相关举措会招致中方的反制。

企业的逻辑很简单：我在中国有工厂、有市场、有供应链，你欧盟要加税，中国反制的时候倒霉的是我。德国车企在中国市场的利润占全球总利润的三四成，一旦中国市场出问题，德国总部就得裁员。

德国交通部长维辛在上海的表态更直接：“关税是一种破坏性的做法，我呼吁欧盟委员会不要寻求加征关税，而要争取良好和公平的竞争规则”。

一个国家的部长在另一个国家的城市里批评自己所属的联盟，这种事放在以前是不可想象的。但2026年的德国就是这么干的。不是德国政府突然“亲中”了，是德国企业逼着政府必须这么做。

这次重启机制还有一个细节特别值得注意：双方同意立即启动出口管制技术层面磋商。

出口管制是欧盟手里的一张牌，也是中国手里的一张牌。欧盟在高端芯片、半导体设备上卡中国，中国在稀土、镓、锗上卡欧盟。

两边互相卡，最后谁疼？当然是德国最疼。因为德国是制造业大国，高端芯片需要进口，稀土也需要进口。两头都被卡，德国的工厂怎么转？

所以德国急着跟中国谈出口管制。不是为了帮欧盟谈，是为了给自己谈。德国想知道中国的出口管制到底会怎么走，德国也好提前做准备。

中国也想通过德国这个渠道，把自己的关切传达到欧盟内部。出口管制磋商这个口子一开，等于中德之间多了一条直接对话的通道，不用什么事都通过欧盟那个慢吞吞的机制。

欧盟对中国电动车磨刀霍霍的时候，德国悄悄跟中国重启了一个停了十年的机制。这不是背叛，这是自救。2530亿欧元的贸易额在那摆着，5200家企业的饭碗在那搁着，德国不可能跟着欧盟一起往悬崖下跳。

联委会重启了，两个工作组设了，出口管制也开始谈了。德国在欧盟的框架内给自己修了一条“单线联系”中国的管道。这条管道能走多远，取决于欧盟的对华政策会硬到什么程度，也取决于德国的企业能撑到什么程度。

但有一点是确定的：当整个欧盟都在往对华强硬的方向冲的时候，德国已经在悄悄找退路了。

这条退路能不能走得通，没有人能打包票。但至少德国已经迈出了第一步。而这一步，比欧盟任何一份声明都更能说明问题。
