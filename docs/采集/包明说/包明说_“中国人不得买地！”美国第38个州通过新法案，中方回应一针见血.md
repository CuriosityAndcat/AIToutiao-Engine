如果告诉一百年前的美国人，将来有一天他们要用法律禁止中国人买地，那帮人大概会反问：中国人不是正在帮我们修铁路吗？

可历史就这么荒诞，一百年前中国人被请来修路，一百年后中国人被赶出农场。连美国华裔议员都忍不住翻出老账本说，这不就是1913年那部《外籍人土地法》的翻版吗？到底谁是威胁，谁在害怕，这个问题得先理清楚。

北卡罗来纳州州长乔希·斯坦，日前签了一堆法案，有打击加密货币诈骗的，有提升机动车管理部门办事效率的。

混在其中的还有一项，禁止与中国政府有关联的个人、企业及其他实体购买北卡州的农业用地以及靠近军事设施等敏感区域的土地。支持者说这是为了"维护国家安全"。

州长签完字就去忙别的了，这件事在他的日程表上大概只占了一行。

但如果把这行字放大，你会看到一幅更大的图景。北卡是第38个推进或修订此类法案的州。从2021年到现在，这股风已经刮了五年，覆盖了美国四分之三以上的州。

佛罗里达和得克萨斯带头，印第安纳、俄亥俄、犹他、威斯康星跟进。2026年5月，美国国会众议院"美中战略竞争特别委员会"主席提出法案，要求严格审查中资在敏感区域购地行为。

从州一级到联邦层面，一条针对中国土地投资的封锁线正在从点连成面。

这不是什么新鲜事。1913年加利福尼亚州颁布《外籍人土地法》，禁止没有公民资格的亚洲移民拥有土地。

一百多年后，同样的逻辑换了包装重新登场，连美国华裔联邦众议员赵美心都直言，此类立法是19世纪《外籍人土地法》的翻版。当年针对日本移民，今天针对中国投资者。

翻看那些法案的具体条款，你会发现一个有趣的现象。它们的打击面宽得惊人，但精确度却低得可怜。北卡的法案禁止"与中国政府有关联"的实体购地。

印第安纳州的SEA 256法案把中国、俄罗斯、古巴、伊朗、朝鲜、委内瑞拉全列进去了。得克萨斯的SB 17法案同样覆盖中国、俄罗斯、伊朗、朝鲜。

这些法案写的都是"敌对国家的公民和企业"，可一个在得克萨斯州生活了二十年的华裔永久居民，跟北京的政策制定者之间到底有什么关系？法案不问这些。

更耐人寻味的是，这些法案在"国家安全"的旗号下，悄悄干了一件跟安全毫无关系的事。他们把"中国"和"威胁"焊在了一起，然后让每一个长着中国面孔的人来承担这个焊点的重量。

法案不会区分你拿的是中国护照还是美国绿卡，不会区分你来美国是投资建厂还是经营农场，不会区分你买地是为了种粮食还是为了建仓库。

在法律的文本里，你只有一个身份：来自"敌对国家"的人。这跟一百年前《外籍人土地法》禁止日本移民拥有土地的逻辑，有什么区别？

那中国投资者真的买走了美国多少地？美国农业部2025年的数据显示，外国实体持有的美国农业用地仅占3.1%。中国投资者持有的比例更是微乎其微。

可就是这微乎其微的比例，已经足够让38个州的议会忙得不可开交。他们花大量时间起草法案、辩论、投票、签字，就为了挡住那几乎看不见的"威胁"。

这跟一个人花三个月时间在自家门口修一堵墙，只为了挡住一只蚂蚁，有什么区别？

更荒唐的是，这些法案在操作层面几乎无法执行。什么叫"与中国政府有关联"？是持股超过一定比例的中国企业算，还是只要有中国公民参股就算？

如果一个中国投资者通过美国注册的空壳公司购地，算不算"与中国政府有关联"？如果一位华裔美国公民替在中国的亲戚代持土地，算不算违规？

这些问题在法案的文本里全都没有答案。立法者只管喊口号，把具体的执行难题扔给了执法部门。执法部门面对这些模糊条款，要么选择性执法，要么干脆不执行。

一部法律要是连执法者都不知道该怎么用，它真正的效果就只剩下一个：制造恐惧。

这股浪潮正在反噬美国自己。中国外交部发言人的回应一针见血：泛化国家安全概念，将经贸投资问题政治化的言行违背市场经济原则和国际贸易规则，有损外界对美国市场环境的信心。

当一个国家开始用"国家安全"来解释一切的时候，"国家安全"这个词本身就不再有任何说服力了。今天禁止买地，明天禁止投资，后天禁止合作。

投资者看到的是一个不可预测的市场，是一个随时可能以"安全"为名没收你财产的国家。谁还敢来？

佛罗里达州SB 264法案2023年通过后，多名居住在佛州的中国公民集体起诉州政府，指控此举构成"购房歧视"。2025年11月，联邦第十一巡回上诉法院裁定允许佛州执行该法律。

法院说可以，但市场说不行。当法律和市场开始说两种语言的时候，受伤的是整个经济体。佛罗里达的房地产经纪人已经感受到寒意，中国买家在佛州的购房咨询量在法案通过后下降了超过四成。

那些本来打算来佛州投资、开公司、创造就业的中国资本，转头就去了加拿大和澳大利亚。佛罗里达以为自己在保护安全，实际上正在把真金白银往外推。

美国已经有38个州推进或修订了此类法案。北卡是2026年7月8日签的字。下一个是谁？不知道。但能确定的是，每一扇关上的门都在提醒世界：这个地方正在变得不那么欢迎你。

他们以为自己在保护国家安全，实际上正在一点点拆掉自己亲手搭建的开放市场。当年他们用开放吸引了全世界的资本和技术，如今他们用封闭把资本和技术往外推。

历史很少直接惩罚做错事的人，它只会让做错事的人慢慢失去原本拥有的东西。而失去的东西，从来不会在法案的签字仪式上被计算进去。

这场始于佛罗里达、蔓延到38个州的"禁止中国买地"运动，还有一个让人后背发凉的后果。它正在把"中国"这个词变成一个有毒的标签。

当你在美国任何一州提起"中国投资者"，别人想到的不再是"创造就业的商人"，而是"威胁国家安全的间谍"。标签一旦贴上，想撕下来就难了。

今天针对的是中国资本，明天针对的可能是中国学生，后天针对的可能是中国游客。这条滑坡的终点在哪里，没有人知道，但走过的每一步都在加速滑向更陡的坡段。

北卡州长签完字就去忙别的了。他签的是一份他觉得无关紧要的行政文件，但在他转身离开的那一刻，美国第38个州的门关上了。

这扇门关上的时候很安静，没有引起太多注意。但当第39个、第40个、第41个州也关上同样的门时，世界会突然发现，那个曾经张开双臂欢迎全世界的美国，已经把自己关在了一间四面都是墙的屋子里。

而墙，从来都是自己砌的。
