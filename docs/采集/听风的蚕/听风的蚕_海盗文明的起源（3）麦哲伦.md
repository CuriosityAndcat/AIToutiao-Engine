上集我们聊到哥伦布，知道这家伙可不是什么好鸟，他简直为海盗文明打下样板，堪称殖民者的祖师爷。所谓因果循环，恶人自有恶人磨，最终哥伦布落得个不得安生的下场。

实际上，在哥伦布第三次远航的时候，西班牙王室就不信任他了，同时派出多个冒险家出海，收获不少，后来天主教双王先后离世，但海洋文化并未消失，西班牙成了无数冒险家的圣地。今天要说的，就是继哥伦布之后，对西班牙最终成为日不落帝国产生重大影响的麦哲伦。

麦哲伦比哥伦布小29岁，1480年出生在葡萄牙北部一个破落的骑士贵族家庭，10岁的时候爹妈都没了，后来去里斯本宫廷当了侍从，16岁又进了葡萄牙国家航海事务厅，25岁那年，他跟着军队去非洲打殖民战争，结果战场上一只脚受了伤，落下个跛脚的毛病，这也为他在环球航行途中丧命埋下了伏笔。

33岁的时候，麦哲伦兴冲冲地向葡萄牙国王曼努埃尔申请组织船队搞一次环球航行，结果被一口回绝了，为啥呢？因为国王觉得东方贸易已经攥在手里了，根本没必要再费劲开辟新航道。1517年10月20日，碰了一鼻子灰的麦哲伦离开了葡萄牙，转头投奔了老对手西班牙。半年后，西班牙国王查理五世，也就是上集提到的伊莎贝拉女王的曾孙，亲自接见了他，麦哲伦赶紧提出航海请求，还献上了一个自己亲手做的彩色地球仪。

国王那叫一个高兴，颇有其曾祖母的风范，当场就批准了计划。要知道西班牙当时正愁没法和葡萄牙抢香料贸易这块肥肉呢，于是准备了一年多，在1519年8月10日，麦哲伦领着5艘船、265号人从西班牙塞维利亚出发了。记住，这船队里头，旗舰是特里尼达号，另外4艘分别是圣安东尼奥号、康塞普西翁号、维多利亚号和圣地亚哥号，船队一头扎进茫茫大西洋，一场伟大的环球航行就这么开始了。

葡萄牙国王一听这消息，瞬间感觉受到了威胁，立马派了一支船队去追击，麦哲伦脑子转得快，一番巧妙躲避，把追兵远远甩在了身后。一个半月后，船队就到了西班牙的属地加纳利群岛，紧接着12月就横渡了大西洋，摸到了南美洲附近的海域，因为当时巴西是葡萄牙的殖民地，麦哲伦只能绕着走，12月13日，船队抵达里约热内卢湾，在这儿稍作补给一番，一个月后，船队开到一个一眼望不到头的大海湾，船员们当时就乐疯了，以为到了美洲的尽头，结果后来才发现，这不过就是个拉普拉塔河口。

这时候团队开始有点骚乱，但还不碍事，又过了3个月，在次年3月31日这一天，南美洲到了隆冬季节，麦哲伦只能带着船队开进圣胡利安港准备过冬，可这时候麻烦来了，天气冷不说，粮食还快见底了，船员们一个个垂头丧气，船队内部直接爆发叛乱，5艘船里有3艘船的船长联合起来反对麦哲伦。

这才到哪呀，麦哲伦自然是不甘心。他故意派人送去一封假意谈判的信，趁对方放松警惕，直接把叛乱的船长给刺杀了，轻松平息了叛乱。

船队往南走，又过了两个月，5月22日，圣地亚哥号在附近海域勘测的时候，遇上风暴迷了路，船员倒是都救回来了，可船却没了。这是船队的第一次重大损失。好在没过多久，麦哲伦在圣胡利安港找到了鱼类和淡水，粮食危机暂时解除。

10月21日，他们在南纬52度的地方发现了一个假角，给它取名叫维尔格内斯角，看到海峡南岸的土著居民日夜烧着篝火，就跟欢迎的仪仗队似的，麦哲伦直接把这块陆地命名为火地岛。

本以为这是个好的开始，可没想到，圣安东尼奥号竟然偷偷溜了，掉头跑回了西班牙，这是第二次遭遇重大损失。但麦哲伦征服地球的信心并没有受损，这时候，船队不小心误入一个大海湾，穿过汹涌的风暴和滔天巨浪，居然发现了一条狭长曲折的海峡，这就是后来大名鼎鼎的麦哲伦海峡。

在这条海峡穿梭了一个月，船队总算驶出了海峡西口，眼前一下子出现了一片浩瀚无际的海洋，在接下来的100多天里，船队在这片海上航行，居然风平浪静，起名小行家麦哲伦一高兴，就给这片海起了个吉祥的名字——太平洋。

这时候，对麦哲伦来说，好消息是终于绕过了南美，坏消息是这太平洋好像比想象中的要大，而且还没有什么能补给的岛屿，船队在这儿漂着，长期缺水果缺蔬菜，好多人都患上了败血症，船上的饮水臭得没法喝，饼干都碎成了粉，里面还爬满了蛆虫，为了填饱肚子，船员们只能把皮革烤了吃，这趟漫长的太平洋之旅，差点就让船队全军覆没。

1521年3月，船队总算熬到了马里亚纳群岛，岛上的土著人特别热心，送来了粮食、水果和蔬菜，船员们感动得不行，可这些土著人一辈子没见过船上的东西，对啥都好奇，就顺手从船上搬走了一些物品，麦哲伦当场就火了，带着人冲上岸，开枪打死了7个土著人，还抢走了好多食物和水，还一把火烧了土著人的家园，还给这岛起了个难听的名字叫强盗岛。事实上，土著人有没有偷东西是无从考证的事。但麦哲伦抢了岛上的食物和水，却是绝对真实的，谁才是强盗，谁知道呢？

3月16日，船队到达菲律宾的霍门洪岛，麦哲伦和他的同伴们，总算是完成了人类历史上第一次横渡太平洋的壮举，也证实了美洲和亚洲之间，确实隔着一片辽阔的水域。3月28日，麦哲伦到了马萨瓦岛，在当地岛民的指引下，4月7日来到了菲律宾中部的宿务港。麦哲伦眼珠子一转，就跟宿务的首领说，咱俩交个朋友呗，只要你承认是西班牙国王的属臣，我就给你提供军事援助，说着还搞了一场军事演习吓唬人家。

宿务首领吓得赶紧答应，一星期后，带着全家老小和几百名臣民接受了洗礼，没几天，这个岛和附近岛上的好多居民，都跟着信了教，麦哲伦一下子就成了这些新基督徒的靠山。可附近马克坦岛上的酋长拉普是个硬骨头，死活不答应，麦哲伦哪受得了这气，于是在1521年4月27日，带着60多个人，坐着3艘船就杀向了马克坦岛。

谁知道浅滩上全是礁石，船根本靠不了岸，麦哲伦只能带着一部分船员涉水登陆，可他们刚上岸，就撞见了拉普率领的岛民，人家早就严阵以待了。麦哲伦赶紧命令火炮手和弓箭手开火，可岛民们根本不怕，嗷嗷叫着就冲了上来，船员们哪里抵挡得住，只能边打边退。

麦哲伦急眼了，下令烧毁岛上的村庄，想扰乱岛民的军心，可岛民们看到自己的房子被烧，更来气了，追着他们猛打。麦哲伦本来就跛脚，跑起来哪有别人快，很快就被标枪刺中，紧接着就被追上来的岛民刺死了！

这个被历史书大书特书的人物，就这么稀里糊涂地陨落了。麦哲伦死后，他的妻弟巴尔波查临时当了老大，巴尔波查还想跟马克坦人要回麦哲伦的尸体，结果人家回了一句，魔鬼的尸体要用来祭祀祖先，绝不可能还给敌人。没过多久，巴尔波查也被胡马邦诱杀了，这胡马邦正是最开始皈依基督教的宿务岛首领。

5月1日，船队狼狈地逃离了宿务港，破旧的康塞普西翁号实在没法带了，只能遗弃烧毁，这时候船队只剩下两艘船，110多个人凄凄惨惨地赶到了文莱，这一路上，船队换了好几任领袖，还闹了好几次叛乱。11月6日，他们总算熬到了香料群岛。11月8日，船队用一些廉价的玩意儿，换来了大批的香料，丁香、豆蔻、肉桂堆得船舱满满当当。

就在他们准备返航西班牙的时候，发现特里尼达号坏得厉害，必须大修，于是埃斯皮诺萨带着50多个人留下，打算等船修好后，从太平洋东路返回西班牙。11月21日，埃尔卡诺领着维多利亚号上的60个人，正式踏上了归途。

为了躲避葡萄牙的船队，维多利亚号一路上不敢停留，不敢靠岸，结果好多船员又因为坏血病死在了路上，横渡印度洋的时候，代价更是惨烈，最后只剩下35个人。

而刚才说的另一艘大修的船特里尼达号，1522年4月6日，修缮完毕后驶离香料群岛的蒂多雷岛，计划横渡太平洋返回西班牙。不料航行途中，船队向北驶至日本以东北纬42-43度海域时，遭遇持续风暴侵袭；再加上坏血病肆虐，船员折损过半。

剩余船员已无力继续续航，只得掉头返航，于当年10月再度回到香料群岛。可他们刚抵达特尔纳特岛，就被葡萄牙海军俘获，船员全被投入监狱。这场归途最终成了劫难，最后仅有4人侥幸存活，数年后才得以返回西班牙，此为后话。

再看维多利亚号这边，1522年5月20日绕过了非洲南端的好望角，7月9日因为船只损坏，只能停靠在葡萄牙领地佛得角群岛，结果被葡萄牙人发现，又抓走了13个人，埃尔卡诺带着剩下的人赶紧跑路，半路上又有4个船员没撑住死了。1522年9月5日，维多利亚号总算摇摇晃晃地回到了西班牙，这时候船上只剩下区区18个人。

从1519年8月到1522年9月，麦哲伦和他的船员们，整整花了三年时间，完成了人类历史上第一次环球航行，他们十不存一，用生命，证明了地球真的是圆的，对此，客观讲，麦哲伦确实为航海学和地理学做出了巨大的贡献，为后来的贸易全球化打下了基础，可同时呢，他也加快了殖民主义的扩张步伐，从此全球进入了海盗文明的时代。那么后来西班牙为何快速登顶之后又快速陨落呢？这一点跟当下的老美何其相似？我们下段接着说。
