# 包明说 仿写风格指南

## 一、风格画像（一句话定位 + 核心特征）

**一句话定位**：用“反差+悬念”制造钩子，以“设问-论证-收束”为骨架，靠“短句节奏+数据佐证+讽刺升华”驱动情绪，专攻国际时政与人物传奇的硬核叙事。

**核心特征**：
- **反差叙事**：标题必含“不是…而是…”“最X的是”等对比结构，制造认知冲突（如“14岁扛枪闹革命，74岁当总统”）。
- **短句节奏**：句子平均长度控制在15字以内，多用句号断句，形成“哒哒哒”的机枪感（如“然后死了，然后内脏没了，然后遗体被送回来了，然后什么都没有了”）。
- **数据佐证**：每个论点必配具体数字、时间、法案编号（如“47年黑名单”“1000万通缉犯”“12人小组”）。
- **讽刺升华**：结尾必用一句反讽或哲理句收束，把情绪从愤怒/震惊拉到“看透了”的高度（如“而墙，从来都是自己砌的”）。
- **口语化但不过度**：用“最魔幻的是”“更离谱的是”等口语词，但避免网络梗，保持新闻评论的严肃底色。

## 二、标题公式（给出 2-3 个可套用模板，并举例）

**公式1：反差+悬念型**  
`[身份/事件A] + [动作/结果B]，[反差点] + [悬念词]`  
- 示例：《14岁扛枪闹革命，74岁当总统后首访中国，这位“老妈妈”不简单！》  
- 示例：《法国人彻底破防！技术是我们教的，结果中国2周就干完法国2年的活》

**公式2：震惊+提问型**  
`[震惊事件] + [提问/反问]`  
- 示例：《器官竟全被掏空！印度船员命丧委内瑞拉，印度誓要委方给个交代？》  
- 示例：《美军12人小组进驻台军营区！名为联合操演，实为给谁“递刀子”？》

**公式3：数字冲击+反转型**  
`[数字] + [动作] + [反转结果]`  
- 示例：《特朗普和朱大帅握手！47年黑名单说撕就撕，1000万通缉犯成好朋友》  
- 示例：《被中国制裁的荷兰大臣来华开口就要彻底了断美国那关他能过》

## 三、开篇模板（2 个可直接套用的开篇句式）

**模板1：设问式开篇（制造悬念）**  
“你有没有想过，[一个反常识的现象]？[补充背景]。”  
- 示例：“你有没有想过，一个14岁扛枪闹革命的少女，74年后会成为一国总统，并且首访就选中国？这位‘老妈妈’的故事，比你想象的更传奇。”

**模板2：反差式开篇（直接抛冲突）**  
“[事件A]！[事件B]？[核心反差]。”  
- 示例：“法国人彻底破防了！技术是我们教的，结果中国2周就干完了法国2年的活。这种差距不是技术问题，是体制问题。”

## 四、叙事结构（总结其最常用的结构，给出骨架）

**标准结构：设问-论证-收束（七层递进）**

1. **钩子**（1段）：用标题中的反差/震惊点，抛出问题或现象  
2. **背景**（1-2段）：用时间线或数据铺陈，解释“为什么这事值得关注”  
3. **拆解**（3-5段）：分层论证，每层用“不是…而是…”“最X的是”“问题在于”等句式引导  
4. **对比**（1-2段）：用历史/他国案例做反差（如“跟一百年前《外籍人土地法》有什么区别”）  
5. **情绪高潮**（1段）：用短句+数据+讽刺，制造愤怒/震惊/讽刺的峰值  
6. **升华**（1段）：用哲理句或反问收束，把情绪拉到“看透本质”的高度  
7. **互动**（1句）：用“你怎么看？”或“欢迎留言”收尾

## 五、金句库（提炼 5-8 个该作者高频句式模板）

1. **“不是…而是…”**（揭示本质）  
   - 示例：“不是口号，是种子、是技术、是实实在在的合作。”  
   - 示例：“他们以为自己在保护国家安全，实际上正在一点点拆掉自己亲手搭建的开放市场。”

2. **“最X的是”**（强调关键点）  
   - 示例：“最引人注目的是，它同时取得了中美欧三张适航证。”  
   - 示例：“最让日本头疼的是：沟通渠道断了。”

3. **“从…到…”**（时间线对比）  
   - 示例：“从14岁的革命少女到74岁的国家元首，整整六十年。”  
   - 示例：“从引进到自主，从‘奖状’到运-12F，从填补空白到全球领先。”

4. **“比…都…”**（夸张对比）  
   - 示例：“比从任何政客嘴里说出来都有分量。”  
   - 示例：“这剧情反转得比好莱坞剧本还离谱。”

5. **“如果…呢？”**（假设引导思考）  
   - 示例：“如果他们没这么做呢？”  
   - 示例：“如果这12个人来自美军‘绿色贝雷帽’特种部队，答案可能跟你想象的不太一样。”

6. **“问题在于”**（转折批判）  
   - 示例：“问题在于，普京希望的‘结束’和泽连斯基希望的‘结束’，是一个意思吗？”

7. **“…，然后…，然后…，然后…”**（递进式短句）  
   - 示例：“然后死了，然后内脏没了，然后遗体被送回来了，然后什么都没有了。”

8. **“嘴说不如船跑”**（口语化行动宣言）  
   - 示例：“嘴说不如船跑，道理讲不通的时候，行动就是最响亮的语言。”

## 六、人称与互动（对读者怎么称呼、怎么收尾互动）

**人称**：无直接称呼（不用“亲爱的读者”），但用“你”引导思考或假设：  
- “你有没有想过…”  
- “如果你只把它当成…那就太天真了”  
- “有人会问：这跟我们有关系吗？”

**收尾互动**：  
- 用反问或升华句收束，不加“点赞转发”等直接呼吁，而是用“你怎么看？”或“欢迎在评论区留下你的观点”自然引导。  
- 示例：“当一个国家开始用‘国家安全’来解释一切的时候，‘国家安全’这个词本身就不再有任何说服力了。你怎么看？”

## 七、情感曲线（典型情绪推进路径）

**路径1：好奇 → 震惊 → 讽刺 → 升华**（适用于国际政治类）  
- 示例：《美俄总统今年第4次通话》  
  好奇（开篇悬念）→ 震惊（通话细节）→ 讽刺（欧洲被晾一边）→ 升华（“墙，从来都是自己砌的”）

**路径2：震惊 → 愤怒 → 悲哀 → 升华**（适用于悲剧/不公类）  
- 示例：《器官竟全被掏空》  
  震惊（开篇细节）→ 愤怒（印度不作为）→ 悲哀（公民遗体都护不住）→ 升华（大国尊严从何谈起）

**路径3：好奇 → 敬佩 → 务实 → 信任 → 希望**（适用于人物/科技类）  
- 示例：《14岁扛枪闹革命》  
  好奇（反差人设）→ 敬佩（传奇经历）→ 务实（合作细节）→ 信任（不是口号是种子）→ 希望（未来展望）

## 八、去AI味要点（如何写得像他而不是机器）

1. **拒绝长句**：每句不超过20字，多用句号断句。AI爱用逗号串长句，要手动拆成“短句+短句+短句”节奏。  
2. **加口语词**：在关键位置插入“最魔幻的是”“更离谱的是”“你猜怎么着”等口语化连接词。  
3. **数据密集**：每个段落至少出现1个具体数字（年份、人数、百分比），避免“很多”“大量”等模糊词。  
4. **讽刺收尾**：结尾不要总结陈词，要用一句反讽或哲理句收束（如“而墙，从来都是自己砌的”）。  
5. **避免模板化开头**：不要用“近日”“据报道”等新闻通稿式开头，直接用设问或反差句。  
6. **情绪递进**：不要平铺直叙，每段都要比前一段情绪更强（震惊→愤怒→讽刺→升华）。  
7. **少用“我们”**：包明说几乎不用“我们”作为主语，多用“你”或直接陈述事实。

## 九、仿写检查清单（写完后对照的自检 5 条）

- [ ] **标题是否包含反差或数字？**（如“14岁 vs 74岁”“47年 vs 1000万”）  
- [ ] **开头是否用设问或反差句？**（避免“近日”“据报道”）  
- [ ] **每段是否有具体数据？**（至少1个数字，如“12人”“2周”“85%”）  
- [ ] **是否有“不是…而是…”或“最X的是”等金句模板？**（至少2处）  
- [ ] **结尾是否有讽刺或哲理升华？**（避免“综上所述”，用“而…从来…”或“你怎么看？”）
