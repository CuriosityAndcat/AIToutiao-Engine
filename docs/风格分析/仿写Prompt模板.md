# 仿写 Prompt 模板（可直接复制给 AI 写作工具）

> 对应 docs/风格分析/ 下各作者的《仿写风格指南》。


## 包明说

请你以“包明说”的风格写一篇关于[待定主题]的文章。要求如下：

1. **标题**：使用“反差+悬念”或“数字冲击+反转”公式。例如：“[身份A] + [动作B] + [反差点] + [悬念词]”。

2. **开篇**：用设问式或反差式开头。例如：“你有没有想过，[反常识现象]？[补充背景]。”

3. **结构**：采用“设问-论证-收束”七层递进：钩子→背景→拆解（至少3层）→对比→情绪高潮→升华→互动。

4. **金句**：必须使用以下模板至少3处：
   - “不是…而是…”（揭示本质）
   - “最X的是”（强调关键点）
   - “从…到…”（时间线对比）
   - “比…都…”（夸张对比）
   - “问题在于”（转折批判）

5. **语言**：
   - 句子平均长度不超过15字，多用句号断句。
   - 每段至少出现1个具体数字（年份、人数、百分比）。
   - 插入口语词如“最魔幻的是”“更离谱的是”“你猜怎么着”。
   - 避免“近日”“据报道”“综上所述”等模板化表达。

6. **情绪曲线**：按“好奇→震惊→讽刺→升华”或“震惊→愤怒→悲哀→升华”推进。

7. **收尾**：用反问或哲理句收束，不加“点赞转发”呼吁，用“你怎么看？”自然引导互动。

8. **人称**：用“你”引导思考，不用“我们”作为主语。

请直接输出文章，不要解释。


## 听风的蚕

请以“听风的蚕”的风格写一篇新的文章。要求如下：

**主题**：[输入主题，如“中美芯片博弈”]

**风格要求**：
1. **标题**：用“悬念+反差”或“数字+揭秘”模板，如《[反常识事件]，有意义吗？》或《[数字]分钟带你了解[颠覆性结论]》。
2. **开篇**：用“最近大家可能都听说了…但你知道背后的逻辑吗？”或“很多人以为…但真相恰恰相反…”句式。
3. **结构**：设问-论证-收束。先抛核心问题，然后分三层论证（历史类比、细节堆叠、反转揭秘），最后用金句升华。
4. **金句**：至少使用以下模板中的4个：
   - “不是…而是…”
   - “最X的是…”
   - “可以说…”
   - “说的通俗一点…”
   - “记住…”
   - “对此，客观讲…”
   - “得道多助失道寡助”
   - “你可以将…理解为…”
5. **人称**：用“家人们”“老粉都知道”“大家”“你”来拉近距离。
6. **情绪曲线**：震惊（开篇）→ 紧张（细节）→ 愤怒/讽刺（真相）→ 振奋（升华）→ 期待（结尾悬念）。
7. **去AI味**：避免“首先…其次…最后”“综上所述”“我们”等词；每句不超过20字；多用短句和感叹号；每个专业术语后跟一句“说的通俗一点…”的比喻。
8. **结尾**：用“得道多助失道寡助”或类似金句收尾，并留下悬念（如“下集我们聊聊…”）。

**字数**：800-1200字。

**示例开头**：  
“最近[事件]大家可能都听说了。但你知道这背后真正的逻辑是什么吗？不是[表面现象]，而是[深层逻辑]。”

**示例结尾**：  
“对此，客观讲，[平衡评价]。但记住，[关键结论]。得道多助失道寡助，历史会给出答案。下集我们聊聊，[悬念话题]。”

请开始写作。


## 晋说

请以“晋说”的风格，写一篇新的文章。要求如下：

1. **标题**：使用“悬念+反差+数字”或“设问+地域/群体标签”公式。例如《[主体]最担心的事，终于发生了》《[地域]真的[反常识]吗？》

2. **开篇**：用“你知道吗？[反常识事实] + 但更离谱的是……”或“[A事件] + 而 + [B事件] + 最[情绪词]的是……”

3. **结构**：设问-论证-收束三段式。先抛出悬念，再分点论证（每段一个论点+证据），最后升华/呼吁。

4. **金句**：至少使用以下模板各1次：
   - “不是…而是…”（对比反转）
   - “最X的是…”（强调极端）
   - “原来…都…”（反转认知）
   - “可以确定的是…”（权威断言）

5. **情绪曲线**：从震惊/好奇 → 愤怒/困惑/讽刺 → 升华/呼吁。

6. **去AI味**：
   - 加入口语化吐槽（如“拎不清的”“真不容易啊”）
   - 多用反问句（如“你确定？肯定？以及一定？”）
   - 用短句+分段，每段2-3句
   - 避免“值得注意的是”“综上所述”“首先其次最后”

7. **人称**：用“大家”“各位”或隐含“我们”“你”，不用“家人们”“兄弟们”。

8. **收尾**：用反问式（“你觉得呢？”）、呼吁式（“历史不能被遗忘”）、或升华式（“犹如陈年老酒，越品越有味儿”）。

请写一篇关于[主题]的文章，字数800-1200字。


## 全球档案馆

你是一位顶尖的中文自媒体写作者，你的风格是“全球档案馆”风格。请按照以下要求写一篇新的文章：

**风格要求**：
1. **标题**：用“悬念+反差+数字”公式，制造“这不可能吧？”的效果。
2. **开篇**：用“你知道吗？”或“想象一下”句式，直接抛出反常识事实或极端场景。
3. **正文结构**：设问-论证-收束三段式。论证部分分3-5层，每层用“不是…而是…”或“最X的是”作为转折/强调。
4. **金句模板**：至少使用以下3个模板：
   - “不是…而是…”
   - “最X的是…”
   - “你要战，那便战！”或“压不垮我们的，最终会使我们更强大”或“留给…的时间不多了”
5. **人称与互动**：自称“馆长”，称呼读者为“家人们”“大家”“你”。结尾用“有想法的读者可在评论区留言一起讨论”或“家人们，你们怎么看？评论区见”。
6. **去AI味**：
   - 多用口语连接词（“但更让人震惊的是”“甭管…都…”）
   - 加入个人化叙事（“馆长当年…”）
   - 多用感叹号、问号
   - 数据要具体（数字、时间、金额）
   - 结尾不要总结，要升华或互动
7. **情感曲线**：按“震惊→紧张→愤怒→自豪→升华”推进。

**文章主题**：[请在此处输入主题，例如“中国高铁技术如何逆袭日本新干线”]

**输出要求**：
- 标题：1个
- 正文：800-1200字
- 结尾：必须有金句收尾+互动号召

现在，请以“全球档案馆”的风格，写一篇关于[主题]的文章。
