# 全球档案馆 仿写风格指南

## 一、风格画像（一句话定位 + 核心特征）

**一句话定位**：  
一个用“悬念+数据+情绪”三连击，把军事、科技、历史硬核话题讲得像悬疑故事一样的“馆长”式自媒体。

**核心特征**：
- **悬念驱动**：每篇文章开头必设一个“你绝对想不到”的钩子，把读者从“好奇”拽进“震惊”。
- **数据密集**：不空谈观点，用具体数字（时间、金额、速度、比例）制造真实感和冲击力。
- **情绪递进**：从震惊→紧张→愤怒→自豪→升华，像过山车一样让读者情绪跟着走。
- **口语化+金句模板**：大量使用“不是…而是…”“最X的是”“你要战，那便战”等固定句式，读起来像在听人讲故事，而非读论文。
- **去AI味明显**：85分左右，主要通过个人化叙事（“馆长说”“有小伙问”）、情绪化标点（感叹号、问号连用）、口语化连接词（“甭管…都…”“怎么…就…”）实现。

## 二、标题公式（给出 2-3 个可套用模板，并举例）

**公式1：悬念+震惊+数字**  
`[数字/时间] + [动词/事件] + [结果/反差]`  
- 例：《一小时之内击穿美日斥巨资搭建的反导拦截体系，这绝不是开玩笑》  
- 例：《美国为何放弃第一岛链？兵棋推演：DF-21D二十分钟击沉2万人舰队》

**公式2：反差+悬念+提问**  
`[看似无害/普通的事物] + [反转/危害/真相] + [问号]`  
- 例：《电子烟能有什么坏心思？它只是想要你的命而已》  
- 例：《世界500强排名第二的国家电网，历史竟然这么憋屈》

**公式3：主体+困境/反转+情绪词**  
`[主体] + [怎么了/遭遇了什么] + [情绪词（憋屈/震惊/愤怒/自豪）]`  
- 例：《空客CEO访华拿下百亿订单，波音CEO喊话布林肯：你快去啊》  
- 例：《向死而生的中国半导体行业，美国的制裁恰恰是浴火重生的契机》

## 三、开篇模板（2 个可直接套用的开篇句式）

**模板1：悬念设问式**  
“你知道吗？[一个反常识的事实/数据]。但更让人震惊的是，[更深层的反转]。今天馆长就来聊聊，[主题]到底是怎么回事。”  
- 真实例句：“你知道吗？世界500强排名第二的，不是苹果，不是微软，而是国家电网。但更让人憋屈的是，这家公司历史上竟然被欺负成这样。”

**模板2：场景冲击式**  
“想象一下：[一个极端场景/冲突画面]。这不是电影，这是[真实事件/兵棋推演结果]。而这一切的起因，要从[一个关键事件]说起。”  
- 真实例句：“想象一下：你正在指挥一支2万人的舰队，突然雷达警报响起——20分钟后，你的舰队将被全部击沉。这不是电影，这是美军兵棋推演的结果。”

## 四、叙事结构（总结其最常用的结构，给出骨架）

**核心结构：设问-论证-收束（三段式）**

1. **设问（开篇钩子）**  
   - 抛出反常识问题或极端场景  
   - 制造悬念，吸引读者继续看

2. **论证（主体展开）**  
   - 分3-5个层次递进  
   - 每层用“不是…而是…”或“最X的是”作为转折/强调  
   - 穿插数据、案例、历史对比  
   - 情绪逐步升级（震惊→紧张→愤怒→自豪）

3. **收束（结尾升华）**  
   - 总结核心观点  
   - 用金句收尾（如“你要战，那便战”“压不垮我们的，最终会使我们更强大”）  
   - 互动号召（“有想法的读者可在评论区留言一起讨论”）

**示例骨架（以《一小时之内击穿美日反导体系》为例）**：  
- 设问：你知道反导拦截有多难吗？  
- 论证1：不是…而是…（解释反导原理，制造紧张）  
- 论证2：最X的是…（展示中国突破，情绪转向自豪）  
- 论证3：你要战，那便战！（升华到国家意志）  
- 收束：天网恢恢，疏而不漏。

## 五、金句库（提炼 5-8 个该作者高频句式模板）

1. **“不是…而是…”**  
   - 用法：否定一个常见认知，强调真实/更惊人的事实。  
   - 例：“不是我们占领和登陆蓝方领土，只会打击和清除他们的军事设施。”  
   - 例：“电子烟真正的危害不是尼古丁，而是极大降低了不吸烟人群向吸烟人群转化的门槛。”

2. **“最X的是…”**  
   - 用法：在叙述中突然拔高，制造“更厉害/更讽刺/更意外”的效果。  
   - 例：“最讽刺的是，修建秦岭隧道时从德国进口盾构机的那家公司在2014年的时候被中铁装备收购了。”  
   - 例：“最让人意外的是，这种戒断反应比吸纸烟时有过之而无不及。”

3. **“你要战，那便战！”**  
   - 用法：在情绪高潮时用于表达决心/宣示立场，简短有力。  
   - 例：“你要战，那便战！来而不往非礼也！”

4. **“压不垮我们的，最终会使我们更强大”**  
   - 用法：在逆境或制裁话题中用于升华，传递正能量。  
   - 例：“压不垮我们的，最终会使我们更强大。中国的芯片行业，正在浴火重生。”

5. **“甭管…都…”**  
   - 用法：口语化表达“无论…都…”，增强语气。  
   - 例：“甭管是电子烟还是纸烟，从今往后别想从我这赚走一分钱。”

6. **“怎么…就…”**  
   - 用法：表达惊讶/嘲讽，常用于国际关系话题。  
   - 例：“怎么你制裁人家中国，人家不要你就算了，怎么盟友还不要我了？”

7. **“留给…的时间不多了”**  
   - 用法：制造紧迫感，常用于竞争/危机话题。  
   - 例：“只能说中国市场留给波音的时间不多了。”

8. **“与其…不如…”**  
   - 用法：对比两个选项，强调后者更优。  
   - 例：“与其说美国在追赶，不如说我们在前面等着你。”

## 六、人称与互动（对读者怎么称呼、怎么收尾互动）

**常用人称**：
- **第一人称**：馆长（自称）、我  
- **第二人称**：你、大家、各位、家人们、兄弟们  
- **第三人称**：国家电网、特斯拉、美国（客观叙述对象）

**互动方式**：
- 开篇提问：“你知道吗？”“有小伙问…”  
- 中间引导：“想象一下…”“你可能会问…”  
- 结尾号召：“有想法的读者可在评论区留言一起讨论”“家人们，你们怎么看？评论区见”  
- 口语化收尾：“甭管怎么说，今天先聊到这儿，下期见！”

## 七、情感曲线（典型情绪推进路径）

**标准路径**：  
`震惊（开篇）→ 紧张/憋屈（问题揭示）→ 愤怒（对手/困境）→ 自豪/希望（突破/逆袭）→ 升华（金句收尾）`

**示例（《世界500强排名第二的国家电网，历史竟然这么憋屈》）**：  
1. 震惊：国家电网世界第二？  
2. 憋屈：历史上被外资欺负、被卡脖子  
3. 愤怒：那些年我们被坑得多惨  
4. 自豪：如今逆袭成全球第一  
5. 升华：那些打不倒我们的，终将使我们更强大

**变体（《电子烟能有什么坏心思？》）**：  
1. 震惊：新规来了  
2. 共鸣：馆长自己戒烟经历  
3. 紧张：戒断反应  
4. 愤怒：电子烟危害被掩盖  
5. 升华：决心戒烟  
6. 呼吁：号召读者一起行动

## 八、去AI味要点（如何写得像他而不是机器）

1. **多用口语连接词**：  
   - 不用“然而”“因此”“此外”，用“但更让人震惊的是”“最X的是”“甭管…都…”  
   - 例：不说“此外，电子烟还有另一个危害”，说“最让人意外的是，这种戒断反应比吸纸烟时有过之而无不及”

2. **加入个人化叙事**：  
   - 用“馆长说”“有小伙问”“我当年也…”  
   - 例：“馆长当年戒烟的时候，那叫一个痛苦…”

3. **情绪化标点**：  
   - 多用感叹号、问号、省略号  
   - 例：“你要战，那便战！”“怎么你制裁人家中国，人家不要你就算了，怎么盟友还不要我了？”

4. **数据要具体，不要模糊**：  
   - 不说“很多”“大量”，说“2万人舰队”“20分钟”“2.8倍”  
   - 例：“中国发动机的运行时间是美国的2.8倍”

5. **结尾不要总结，要升华或互动**：  
   - 不用“综上所述”，用“天网恢恢，疏而不漏”“留给波音的时间不多了”“有想法的读者可在评论区留言”

6. **金句模板要自然嵌入**：  
   - 不要生硬套用，要在情绪高点自然出现  
   - 例：在讲完盾构机逆袭故事后，自然接“最讽刺的是，那家德国公司被我们收购了”

## 九、仿写检查清单（写完后对照的自检 5 条）

1. **标题是否用了“悬念+反差+数字”公式？**  
   - 检查：标题是否在3秒内让读者产生“这不可能吧？”的好奇心。

2. **开篇是否用了“你知道吗？”或“想象一下”句式？**  
   - 检查：前50字是否制造了悬念或冲击感。

3. **正文是否至少用了2个金句模板？**  
   - 检查：“不是…而是…”“最X的是”“甭管…都…”是否自然出现。

4. **情绪是否按“震惊→紧张→愤怒→自豪→升华”推进？**  
   - 检查：读者读完后是否经历了情绪起伏，而非平铺直叙。

5. **结尾是否用了金句收尾+互动号召？**  
   - 检查：最后一段是否有“留给…的时间不多了”“压不垮我们的…”等句式，以及“评论区见”等互动。
