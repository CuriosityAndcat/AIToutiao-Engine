# 晋说 仿写风格指南

## 一、风格画像（一句话定位 + 核心特征）

**一句话定位**：  
一个用“悬念+反差”钩子、口语化叙事、情绪递进（震惊→愤怒→升华）来包装深度内容（历史考据、国际分析、文化保护）的“硬核科普+情绪共鸣”型创作者。

**核心特征**：
- **标题即钩子**：悬念（“最担心的事发生了”）、反差（“狗窝装空调 vs 欧洲没空调”）、数字（“30年没造出步枪”）
- **口语化但不油腻**：不用“家人们”“兄弟们”，但用“各位”“大家”，偶尔用“拎不清的阿三”“真不容易啊”等吐槽
- **金句模板化**：高频使用“不是…而是…”“最X的是”“原来…都…”等句式
- **情绪驱动**：开篇震惊/好奇 → 中段愤怒/讽刺/困惑 → 结尾升华/呼吁
- **结构清晰**：设问-论证-收束，或总-分-总，每段有明确论点
- **去AI味强**：有个人吐槽、反问、口语化短句、数据细节、历史典故

## 二、标题公式（给出 2-3 个可套用模板，并举例）

### 模板1：悬念+反差+数字
**公式**：`[主体] + [最担心/最震惊/最离谱] + [数字/事件] + [反差结果]`  
**示例**：  
- 《日本最担心的事，终于发生了》  
- 《一支步枪30年没造出来，哪来的底气说自己是强国？》  
- 《中国的狗窝都装上空调了，欧洲却还有80%的家庭没空调》

### 模板2：设问+地域/群体标签
**公式**：`[地域/群体] + [真的/难道] + [反常识/反直觉] + [疑问词]`  
**示例**：  
- 《山西人不全都说山西话，说山西话的不全都是山西人，这是真的吗》  
- 《山西晋城，世界花园城市，一个不可错过的城市》（隐含反问“叫城区就差了？”）

### 模板3：事件+情绪+升华
**公式**：`[事件/现象] + [情绪词] + [是/不是] + [抽象概念]`  
**示例**：  
- 《混乱，是阶梯》  
- 《从自己揍自己到宣传片翻车，印军那些离谱的操作》

## 三、开篇模板（2 个可直接套用的开篇句式）

### 模板1：悬念式反问+数据/细节
**句式**：`“你知道吗？[反常识事实] + [数据/细节] + 但更离谱的是……”`  
**示例**：  
> “你知道吗？山西人不全都说山西话，说山西话的不全都是山西人。但更离谱的是，这种看似矛盾的现象，背后藏着一段千年历史。”

### 模板2：反差对比+情绪词
**句式**：`“[A事件] + 而 + [B事件] + 最[情绪词]的是……”`  
**示例**：  
> “中国的狗窝都装上空调了，欧洲却还有80%的家庭没空调。最讽刺的是，他们不是买不起，而是法律不允许装。”

## 四、叙事结构（总结其最常用的结构，给出骨架）

### 核心结构：设问-论证-收束（三段式）

**骨架**：
```
1. 设问/悬念（1-2段）
   - 抛出反常识问题或事件
   - 用数据/细节制造冲击
   - 情绪：震惊、好奇

2. 论证/展开（3-5段）
   - 分点论述（可用“第一…第二…”或“最X的是…”）
   - 每段一个核心论点+证据（数据、典故、案例）
   - 穿插口语化吐槽/反问
   - 情绪：愤怒、讽刺、困惑、探索

3. 收束/升华（1-2段）
   - 总结观点
   - 升华到文化/历史/社会意义
   - 情绪：感动、希望、呼吁、冷静提醒
```

**示例骨架（《日本最担心的事，终于发生了》）：**
- 设问：日本最担心的事是什么？→ 不是中国，而是新加坡
- 论证：731部队罪行细节 → 日本掩盖历史 → 国际反应
- 收束：呼吁行动，升华到“历史不能被遗忘”

## 五、金句库（提炼 5-8 个该作者高频句式模板）

### 1. “不是…而是…”（对比反转）
**句式**：`“不是[A]，而是[B]”`  
**示例**：  
- “方言不仅是交流工具，还是地域文化载体。”  
- “不是中国做的，而是一个南亚小国，新加坡。”  
- “不是箭靶，就是‘日’（太阳）。”

### 2. “最X的是…”（强调极端）
**句式**：`“最[形容词]的是，[具体细节]”`  
**示例**：  
- “最欧洲的是，能一本正经的说出这种歪理邪说。”  
- “最著名的例子即为岳飞的《满江红》。”  
- “最讽刺的是，他们不是买不起，而是法律不允许装。”

### 3. “原来…都…”（反转认知）
**句式**：`“原来[我们/大家]都[被/以为]……”`  
**示例**：  
- “原来我们都被日本人骗了。”  
- “原来阿三连一支步枪都造不出来！”

### 4. “如果…就…”（假设推理）
**句式**：`“如果[A]，[就/则][B]”`  
**示例**：  
- “如果是在受保护的城市区域，还得去文物部门申请。”  
- “如果，矢的本意就是箭矢，很难完全合逻辑地引申出上述全部字义。”

### 5. “从…到…”（对比递进）
**句式**：`“从[A]到[B]，[总结]”`  
**示例**：  
- “从自己揍自己到宣传片翻车，印军那些离谱的操作。”  
- “从开篇的震惊，到结尾的讽刺，情绪一路飙升。”

### 6. “可以确定的是…”（权威断言）
**句式**：`“可以确定的是，[事实/结论]”`  
**示例**：  
- “可以确定的是，‘晋’立国是在唐叔虞这一代。”  
- “可以确定的是，这个人不会说普通话，这样说出的方言才更纯正。”

### 7. “不是…而是…+ 反问”（强化反驳）
**句式**：`“不是[A]，而是[B] + 反问句”`  
**示例**：  
- “不是好事，甚至可以说是一个坏消息。但你能怎么办？”  
- “能造核弹，却造不出步枪？你确定？肯定！以及一定？”

### 8. “犹如…越品越有味儿”（诗意升华）
**句式**：`“犹如[比喻]，越[动作]越[感受]”`  
**示例**：  
- “犹如一杯陈年老酒，越品越有味儿。”  
- “犹如一部历史长卷，越读越厚重。”

## 六、人称与互动（对读者怎么称呼、怎么收尾互动）

**对读者称呼**：
- 极少用“家人们”“兄弟们”
- 偶尔用“大家”“各位”（如《混乱，是阶梯》中的“大家”“各位”）
- 大多数情况下无直接称呼，但通过“我们”“你”隐含共情（如“我们都被日本人骗了”）

**收尾互动**：
- **反问式**：“你觉得呢？”“你认同吗？”
- **呼吁式**：“希望之后，[主体]的发展也会越来越好。”
- **升华式**：“历史不能被遗忘，行动从现在开始。”
- **吐槽式**：“拎不清的阿三，真拿他们没办法。”

**示例收尾**：
- “相信之后，晋城的发展也会越来越好。”（《山西晋城》）
- “所以，别再被忽悠了。”（《日本最担心的事》）

## 七、情感曲线（典型情绪推进路径）

### 路径1：震惊→愤怒→讽刺→升华（国际/军事/社会类）
**示例**：《日本最担心的事》《一支步枪30年没造出来》  
- 开篇：震惊（“最担心的事发生了”）  
- 中段：愤怒（731罪行细节、造不出步枪）  
- 转折：讽刺（“一本正经说歪理”）  
- 结尾：升华（“历史不能被遗忘”）

### 路径2：好奇→困惑→惊讶→理解→升华（文化/历史类）
**示例**：《山西人不全都说山西话》《闲言说“晋”》  
- 开篇：好奇（“这是真的吗？”）  
- 中段：困惑（“为什么会有这种矛盾？”）  
- 转折：惊讶（“原来背后有千年历史”）  
- 结尾：理解→升华（“犹如陈年老酒，越品越有味儿”）

### 路径3：中立→委屈/不服→希望→升华（地域/城市类）
**示例**：《山西晋城》  
- 开篇：中立（介绍城市）  
- 中段：委屈（“铁路不通，发展公路”）  
- 转折：希望（“竭尽所能寻找突破”）  
- 结尾：升华（“发展也会越来越好”）

## 八、去AI味要点（如何写得像他而不是机器）

1. **加口语化吐槽**  
   - 不要“值得注意的是”，要“最离谱的是”  
   - 不要“综上所述”，要“拎不清的阿三”  
   - 不要“因此”，要“所以，别再被忽悠了”

2. **用反问句替代陈述句**  
   - 不要“这是一个重要问题”，要“你确定？肯定？以及一定？”  
   - 不要“这让人难以理解”，要“叫城区就能说这个城市差吗？”

3. **插入个人情绪词**  
   - 不要“数据显示”，要“最讽刺的是，数据显示”  
   - 不要“这很令人震惊”，要“真不容易啊，终于找对人了”

4. **用短句+分段**  
   - 不要长段落，每段2-3句  
   - 多用“。”和“？”断句，少用“，”和“；”

5. **加入具体细节/典故**  
   - 不要“历史上有很多例子”，要“考之史籍，晋地数千年来历史政区相对稳定”  
   - 不要“有很多数据”，要“欧洲80%的家庭没空调，而中国的狗窝都装上了”

6. **避免AI常用词**  
   - 避免“值得注意的是”“综上所述”“首先其次最后”  
   - 替换为“最X的是”“可以确定的是”“第一…第二…”
