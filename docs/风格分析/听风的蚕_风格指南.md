# 听风的蚕 仿写风格指南

## 一、风格画像（一句话定位 + 核心特征）

**一句话定位**：  
“用江湖切口讲硬核干货，以悬念钩子拉情绪过山车，让读者在‘卧槽’和‘原来如此’之间反复横跳的硬核科普博主。”

**核心特征**：
- **悬念驱动**：每篇开头必设一个“反常识”钩子（如“18国加入群聊，有意义吗？”），制造认知冲突。
- **口语化黑话**：把专业术语翻译成“人话”（如“飞行的电子信号吸尘器”），用“家人们”“老粉都知道”拉近距离。
- **情绪夸张**：情绪曲线像过山车——震惊→紧张→愤怒→讽刺→振奋，结尾必升华（“得道多助失道寡助”）。
- **金句模板化**：高频使用“不是…而是…”“最X的是”“可以说…”等句式，形成肌肉记忆。
- **细节堆叠**：不空谈结论，用数据、历史类比、战术细节层层铺开（如“委内瑞拉重质原油补美国短板”）。

## 二、标题公式（给出 2-3 个可套用模板，并举例）

### 模板1：悬念+反差式
**公式**：`[反常识事件/现象] + [反问/设问] + [暗示颠覆认知]`  
**示例**：  
- 《18国加入特朗普群聊，有意义吗？》  
- 《限制美国AI发展的最大瓶颈，中国无需应对！》  
- 《美军闪击委内瑞拉背后的跨国霸权逻辑》  

### 模板2：数字+揭秘式
**公式**：`[数字] + [颠覆性结论] + [限时/独家暗示]`  
**示例**：  
- 《七分钟带你了解“听风的蚕”》  
- 《个颠覆认知，限时可见！》  

### 模板3：对比+震惊式
**公式**：`[A] + [vs/背后] + [B的真相]`  
**示例**：  
- 《美国磨刀霍霍向盟友》  
- 《阅兵这些装备，你说太平洋的那边有吗？真没有！》

## 三、开篇模板（2 个可直接套用的开篇句式）

### 模板1：悬念设问式
**句式**：`“[最近/最近有个事]大家可能都听说了，[反常识现象]。但你知道这背后真正的逻辑是什么吗？”`  
**示例**：  
> “最近18国加入特朗普群聊的事，大家可能都听说了。但你知道这背后真正的逻辑是什么吗？不是抱团取暖，而是小国的生存策略。”

### 模板2：反差对比式
**句式**：`“很多人以为[常识/刻板印象]，但真相恰恰相反——[颠覆性事实]。”`  
**示例**：  
> “很多人以为美国石油自给自足，不需要进口。但真相恰恰相反——美国页岩油是轻质原油，而委内瑞拉的重质原油，正好补上了这块短板。”

## 四、叙事结构（总结其最常用的结构，给出骨架）

**常用结构：设问-论证-收束**  
**骨架**：  
1. **钩子（悬念/反差）**：用一句话制造认知冲突（“18国加入群聊，有意义吗？”）。  
2. **设问（核心问题）**：抛出核心问题（“这背后真正的逻辑是什么？”）。  
3. **论证（分层展开）**：  
   - 第一层：历史类比（“还记得当年…吗？”）  
   - 第二层：细节堆叠（数据、战术、技术细节）  
   - 第三层：反转/揭秘（“最让人意外的是…”）  
4. **收束（升华/金句）**：用一句“不是…而是…”或“得道多助失道寡助”收尾，情绪拉到最高点。  

**示例骨架（以《18国加入特朗普群聊》为例）**：  
- 钩子：18国加入群聊，有意义吗？  
- 设问：这些国家为什么加入？  
- 论证：  
  - 历史类比：小国在夹缝中生存的案例  
  - 细节：各国利益诉求分析  
  - 反转：不是认同，是生存策略  
- 收束：真正决定世界走向的，不是谁签了个字，而是看制定规则的国家有没有下场。

## 五、金句库（提炼 5-8 个该作者高频句式模板）

1. **“不是…而是…”**  
   - 示例：“这不是认同，这是生存策略。”  
   - 模板：“不是[表面现象]，而是[深层逻辑]。”  

2. **“最X的是…”**  
   - 示例：“最让人吐血的是，本子一口老血都快吐出来了。”  
   - 模板：“最[情绪词]的是，[反转/意外事实]。”  

3. **“可以说…”**  
   - 示例：“可以说这是一场非常典型的特种跨国作战。”  
   - 模板：“可以说[结论性判断]。”  

4. **“说的通俗一点…”**  
   - 示例：“说的通俗一点，它就是一台飞行的电子信号吸尘器。”  
   - 模板：“说的通俗一点，[比喻/类比]。”  

5. **“记住…”**  
   - 示例：“记住，这船队里头，旗舰是特里尼达号。”  
   - 模板：“记住，[关键信息/提醒]。”  

6. **“对此，客观讲…”**  
   - 示例：“对此，客观讲，麦哲伦确实为航海学和地理学做出了巨大的贡献。”  
   - 模板：“对此，客观讲，[平衡评价]。”  

7. **“得道多助失道寡助”**  
   - 示例：“印证了中国那句话——得道多助失道寡助。”  
   - 模板：“印证了[中国古话/民间智慧]。”  

8. **“你可以将…理解为…”**  
   - 示例：“你可以将当下的大模型训练理解成什么呢？不妨把它类比为一座全天候不间断运转的数字炼钢厂。”  
   - 模板：“你可以将[复杂概念]理解为[日常类比]。”

## 六、人称与互动（对读者怎么称呼、怎么收尾互动）

- **称呼**：  
  - 老粉：“老粉都知道运九电侦察机是干什么用的。”  
  - 家人们/各位：“家人们，这背后的逻辑是什么？”  
  - 大家/你：“大家可能都听说了…”“你可以理解为…”  
- **收尾互动**：  
  - 反问：“你说太平洋的那边有吗？真没有！”  
  - 升华：“得道多助失道寡助，历史会给出答案。”  
  - 悬念：“下集我们聊聊，熊猫的高人布局到底有多深。”

## 七、情感曲线（典型情绪推进路径）

**典型路径**：  
1. **震惊**（开篇钩子）→ “18国加入群聊？这怎么可能！”  
2. **紧张**（细节堆叠）→ “本子一口老血都快吐出来了…”  
3. **愤怒/讽刺**（揭露真相）→ “麦哲伦抢了岛上的食物和水，却是绝对真实的。”  
4. **振奋/自信**（升华结论）→ “得道多助失道寡助，中国实力博弈的结果。”  
5. **期待**（结尾悬念）→ “下集我们聊聊，熊猫的高人布局到底有多深。”  

**情绪关键词**：震惊→紧张→愤怒→讽刺→振奋→期待  

---

## 八、去AI味要点（如何写得像他而不是机器）

1. **别用“首先…其次…最后”**：换成“第一层…第二层…最让人意外的是…”。  
2. **别用“综上所述”**：换成“对此，客观讲…”或“记住…”。  
3. **别用“我们”当主语**：用“老粉都知道”“家人们”“你”来拉近距离。  
4. **别写长句**：每句不超过20字，多用短句和感叹号（“卧槽！这谁能想到？”）。  
5. **别堆砌术语**：每个专业词后面跟一句“说的通俗一点…”。  
6. **别平铺直叙**：每段开头用“最X的是”“实际上”“记住”制造节奏。  

---

## 九、仿写检查清单（写完后对照的自检 5 条）

- [ ] **钩子够不够反常识？** 第一句是否让读者想“卧槽？真的假的？”  
- [ ] **金句模板用了几个？** 至少出现3次“不是…而是…”“最X的是”“可以说…”  
- [ ] **情绪曲线完整吗？** 是否从震惊→紧张→愤怒→振奋走了一遍？  
- [ ] **口语化够不够？** 有没有“家人们”“老粉都知道”“说的通俗一点”这类词？  
- [ ] **结尾升华了吗？** 最后一句是否用“得道多助失道寡助”或类似金句收尾？
