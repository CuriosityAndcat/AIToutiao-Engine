## 根本边界（抄写一遍，签字生效）

- 他人文字不入库（剪藏 / 书摘 / 别人写给我的），哪怕深刻影响过我；
- 纯工作件不入库（会议纪要 / 方案 / 客户项目）；日记里的工作经历可以入；
- AI 生成的分析文档不入库（"关于我的文字"≠"我的文字"）；
- 模糊项不擅自裁决，列入"待裁决"由我逐件定夺。
