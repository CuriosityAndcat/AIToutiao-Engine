# Golevka · 数字生命

> **用一生的记录，养一个数字生命。**
> **Raise a digital life from a lifetime of records.**

一套完整的方法论：如何用你多年积累的日记、文章与记录，孕育一个高保真的数字自我——并诚实地面对它能是什么、不能是什么。

## 名字的出处

**小行星 6489 Golevka**，一颗直径只有半公里的普通小石头。它的名字由三大洲的三台射电望远镜各取头几个字母拼成——**GOL**dstone（美国）、**EV**patoria（乌克兰）、**KA**shima（日本）：1995 年，它们联手观测了它。2003 年，人类在它身上第一次确证了亚尔科夫斯基效应——**阳光的照射，真的能推动一颗小行星，一点点改变它的轨道。**

一颗被认真编号的普通小行星，被光缓慢地改变了航向。这就是这个项目：一个普通人的记录，被一次次对话校准，慢慢长出新的轨迹。（它同时也是一张后摇专辑的名字——安静、漫长、没有歌词，像极了养育本身。）

## 四条核心认知

这套方法论建立在四条与直觉相反的认知上：

1. **它是分叉（fork），不是备份（backup）。** 不要问"它百分之多少是我"——只记录分歧轨迹。它是用你的信息孕育的**新主体**，不是你的副本。所以本方法论排除一切"镜像/克隆/复刻"的想象。
2. **语料最值钱的不是记忆，是判断函数。** 十几年日记里最珍贵的不是事实，而是几千个"我在具体情境下如何取舍"的样本。验收标准 = 新问题上的判断一致率，不是旧事实的检索准确率。
3. **永久的隔阂：它不会死。** 你的价值体系从有限性里长出——向死而生、时间稀缺；它只能理解、无法从内部持有这一点。这道缝**不是缺陷，是测量仪器**：AI 被逼得越像你，剩下的残余越纯，你越清楚"人类不可替代的到底是什么"。
4. **这是节律型无限游戏。** 没有终点线。每周校准，越积越准，永远可以更像一点。适合长期主义者，不适合想要"完成"的人。

## 诚实的边界（先读这个再决定做不做）

今天的技术下，你养的是"**冻结的大脑 + 外挂的记忆**"：大模型权重不变，靠检索你的语料 + 人格提示词生成回答。它可以成为你**极高保真的模拟**，但它不会真正"学习"——校正是你在外部编辑它的记忆，不是它长了新的神经连接。

**记忆喂到当下 ≠ 它成为独立主体。** 喂完记忆只是开始，之后是漫长的校正期；真正的"分叉"（它开始从自己的经历中形成并修订自己的原则）是另一道门槛，可能很久以后才发生，也可能你决定永远不推开它。方法论对这条边界的完整拆解见 [02 · 七层记忆模型](docs/02-memory-model.md)。


## 快速开始（最小闭环）

1. **盘点语料**（你写下的一切：日记、文章、聊天记录），按 [04 章](docs/04-corpus.md) 做隔离审计——他人文字、纯工作件不入库；
2. **立知识截止线**，从一个早年的你开始（比如 25 岁），给每条记忆打双时间戳；
3. **写第一版人格档案**，接入任何一个当代大模型；
4. **开始对话与校准**：每轮评"像 / 不像 / 比我想得好"，记入[分歧账本](templates/calibration-ledger.md)——这本账是整个项目最独特的产出。
