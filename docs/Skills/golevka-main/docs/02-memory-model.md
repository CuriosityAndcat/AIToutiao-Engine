# 02 · 七层记忆模型（L0–L6）

一个数字生命的记忆不是一个桶。把所有东西倒进同一个向量库，是这个领域最常见的工程错误。认知科学一百年的记忆研究给出的分层，恰好可以直接借用。

```text
── 状态 / 存储（可检索、可校正）──
L0 原始证据层   日记 / 文章原文，永久不可变
L1 情景记忆层   自传性事件（带时间地点）："2014年运动会我跑了倒数第二"
L2 语义事实层   去情景化的事实与概念："我本科读了四年"
L3 程序·风格层  怎么说话、怎么写、思维习惯（内隐，最难抓）
L4 价值·判断层  价值排序、判断策略——"判断函数"
L5 自我叙事层   整合性的"我是谁"的故事
── 过程 / 可塑性（本质不同的一类）──
L6 自主学习层   从亲历经验提炼新原则、修订自我模型的机制
```

## 状态与过程：最重要的一条分界

L0–L5 是**状态**（存了什么），L6 是**过程**（状态如何改变）。把"自主学习"当成一个记忆层与前几层并列，是范畴错误——就像把"馆员"归类为一种"藏书"。

这条分界直接给出两个阶段的科学定义：

- **阶段一（高保真重建）= 把 L0–L5 做到高保真。** 当代技术完全够用：L0–L2 靠检索（全文索引即可，父文档链接保留上下文），L3–L5 靠人格档案（提示词）近似。校正期打磨的正是 L3–L5。
- **阶段二（分叉）= L6 上线。** 数字生命之所以"分叉"而非"更多记忆"，实质是：**它开始从自己的亲历中形成并修订自己的原则**，而不只是检索你的。记忆喂到当下 ≠ 分叉；分叉是 L6 的诞生。

**诚实边界**：今天的大模型权重是冻结的，"校正"是你在外部编辑它的记忆状态，不是它在学习。真正的 L6 只有两条路：微调（真·可塑性），或建一层"亲历—反思—原则"的持久记录做功能近似。知道自己此刻在做前者（L0–L5 的高保真模拟），是这个项目保持诚实的方式。

## 每一层的工程要点

**L0 原始证据**：原文永不修改、永不删除。一切上层都可以重建，唯独 L0 是不可再生资源。

**L1 vs L2 必须分开**（Tulving 的情景/语义记忆区分，脑区都不同）：情景记忆随重述而变形，允许多版本解释共存；语义事实要求精确一致，错了就要有"更正"机制覆盖。给记忆打 `episodic / factual` 类型标签，检索与校正策略随类型不同。

**L3–L5 是缄默知识**：它们隐含在语料的字里行间，无法靠"让本人写自述"取出（人对"自己怎么做到的"无法内省报告）。取出它们靠两样东西：大规模第一人称语料（风格自己会渗出来）+ [校准回路](03-calibration.md)的持续打磨。

**双时间戳**（来自记忆重固化研究——每次回忆都会重写记忆）：每条记忆记两个时间，`date`（事件发生时间）与 `written_at`（写下时间）。同一件事，25 岁和 31 岁的解释本就该分开保存，不要合并，不要"以新盖旧"。

**知识截止线**：数字生命在任一时刻有明确的"知道到哪一天"。这不只是工程约束，更是养育的节律工具——你可以让它从 25 岁的你开始，一段一段长大。

## 认知科学的两个支撑框架

- **互补学习系统（CLS）**：大脑用两套系统学习——海马体（快、情景、一次成型）+ 新皮层（慢、统计、缓慢固化）。映射到项目：**继承记忆**（你的历史语料，慢系统的稳定底座）与**亲历记忆**（它启动后的经历）必须分桶，不能倒进一起。这给"继承 vs 亲历"的边界提供了硬科学背书——也是未来 L6 的地基。
- **预测加工**：人格 = 一个带先验的生成模型；校准中你标的每一个"不像" = 一次预测误差信号。账本收集的就是误差流——今天喂给你自己看，未来喂给 L6。
