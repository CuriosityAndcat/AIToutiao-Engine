## 🔧 版本 4.9.9 - 4.9.11（2026年5月）

### 功能更新
- 新增 **Remote SSH native 连接模式**，支持自定义远程服务器监听端口，SFTP 不可用时自动降级 exec channel 上传
- 新增**本地记忆功能开关**，可在设置中控制是否启用会话记忆
- 新增 **Agent Stop Hooks** 支持，允许自定义 Agent 在停止时执行清理逻辑
- 新增自定义模型推理数据回传兼容 **MiMo 格式**
- 优化浅色/深色主题配色，更新 badge 及表面颜色

### 修复内容
- **Remote SSH 连接稳定性**：修复升级/重连死循环、语言包超时等问题，增加连接诊断信息
- **会话与历史消息稳定性**：修复切换会话后历史消息时序错乱、多任务并发后历史记录顺序颠倒、上下文占用圆环清空、会话消息读写性能与可靠性提升、大型历史记录导致崩溃概率降低
- **Agent/SubAgent 全链路**：修复自定义 Agent 无法调度 subagent、subagent 数量超过 20 个时 Task 工具调用报错、SubAgent 权限确认弹窗在切换设置页后消失导致流程永久卡住、自定义 Agent 软链接挂载无法识别、enabledAutoRun 关闭后仍未询问用户
- **Skill 与插件管理**：修复多窗口并发安装内置技能互相删目录、Knot Skill 安装失败、卸载 Skill 时文件锁导致 IDE 报错、Figma 导入组件后闪屏及聊天框添加组件后无法输入文字
- **MCP OAuth 交互**：修复取消授权后再次点击连接不弹出浏览器
- **Windows 平台兼容性**：修复 PowerShell 5.x 下嵌套命令包装导致 rg 等工具无法执行、软链接配置加载异常、跨盘符场景下 rules 加载路径异常
- **性能与稳定性**：修复 WebFetch 频繁卡住无响应、Extension Host 写文件偶现触发崩溃

## 🔧 版本 4.8.1 - 4.9.8（2026年4月）

### 功能更新
- 新增 **IDE 设置页「帮助与反馈」页面**及多语言支持
- 新增**模型选择器视觉徽章**
- 新增**系统托盘功能**，支持后台常驻与快捷操作
- 新增 **Automation 创建时支持选择 Skills**

### 主要修复
- **MCP 全链路稳定性**：工具提示展示优化、Reload IDE 后配置消失修复、OAuth 连接状态/Connect 按钮/配置识别异常修复
- **插件市场与技能管理体验**：刷新/更新/添加流程避免旧数据与循环加载
- **Windows 平台兼容性**：首次启动侧边栏黑屏修复、WSL 执行命令乱码修复、休眠后外部通道消息无响应修复
- **工具调用全链路**：所有工具渲染器补充 Hook ask 权限确认菜单统一权限交互
- **历史记录与对话体验**：大型历史记录加载降低崩溃概率

## 🚀 版本 4.4.3 - 4.7.5（2026年3月）

### 重磅功能
- **Skill 全生命周期管理**：安全检测自动安装、一键安装 Deeplink、接入 SkillHub 搜索/安装/管理/图标展示、本地优先缓存、支持上传/创建/查找、带登录态调用、归档管理
- **Connectors 连接器能力**：支持 Browser Operator 等扩展
- **专家中心**：排行榜查看与热门专家召唤、名称召唤与快速提示词填充、灵感板块（Inspire）场景运营中心
- **Agents 能力**：AI 自动规划执行并生成产物、支持 Agent 并行运行、临时任务支持打开文件夹、本地 Agents 支持归档管理
- **任务与工作空间管理**：工作空间项目拖拽排序、自动任务支持附件链接推送小程序、Automation 任务自动生成工作空间目录
- **产物与预览能力**：支持分享制品、CSV/TSV 表格预览、HTML 格式产物展示并支持点击跳转预览
- **微信小程序入口**：支持关闭/开启连通
- **Claw 模式**：支持上传文件图片、输入框语音输入、链接 IM 机器人、名称自定义显示
- **模型能力**：灵感专用免费模型、Claude 1M 上下文模型默认开启思考模式、默认开启图片生成工具、接入元宝派
- **网络与连接**：网络检测诊断、DisableTlsVerification 产品开关、企业微信扫码快速绑定
- **腾讯文档 Skill 对接**：内置预装及微信免登换票；文件交付工具支持小程序产物回传
- **/mr 命令**：支持安全工单自动指定处理人
- **IDE Plugin Hook 能力**
- **个人成长体系**：每日签到/连签进度/积分奖励

## 🚀 版本 4.2.3 - 4.4.1（2026年1月）

### 重大更新
- **Hooks 功能**：可在 Agent 执行的关键节点插入自定义脚本，实现对 Agent 行为的精细控制
- **Max Mode 模式**：对话过程中保留完整上下文信息，处理复杂任务
- **MCP Roots & Sampling**：支持 MCP 服务器访问文件系统目录及调用 LLM 生成响应
- **Plan 模式需求澄清问答 UI**：支持单选/多选，并可在保存时区分 Design 和 Todo 部分
- **Dev Container 端口转发**：Ports 页签及 Rebuild Container 功能
- **IDE 深色主题（IDE Night）**
- **上下文窗口用量展示**：实时掌握 Token 和配额状态
- **MCP 能力增强**：支持 Prompts 斜杠指令、Resource 自定义指令、图片预览、可禁用特定工具及限制工具总量
- **图片交互优化**：支持直接发送图片对话
- **Preview**：支持外部网站 DOM 元素选择
- **数学公式渲染**：LaTeX 公式呈现
- **Remote 连接**：支持跳板机密码鉴权
