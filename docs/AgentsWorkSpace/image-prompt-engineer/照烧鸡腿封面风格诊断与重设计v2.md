## 二、风格定位总结

**参考图本质 = 智能手机 App 商品详情页截图**：

- 不是海报，是「产品落地页」
- 暗调+暖金（"深夜食堂"既视感）
- 食材是"故事主角"（自上而下的工艺流），菜是"舞台"
- UI 元素（状态栏/导航/徽章）暗示"可购买/可点击"，提高转化暗示

**这正是小红书/抖音美食号封面标准模板**：
- 用户在信息流中看到的第一眼，9:16 全屏沉浸
- "食材故事"暗示工艺考究，"工艺"暗示"值得尝试"
- 底部徽章暗示"卖点提炼"

## 三、新 Prompt 策略（v2 封面）

### 3.1 核心改动

1. **比例 9:16（1024×1792）**
2. **完整复刻手机 UI 框架**（状态栏+X键+底部徽章栏）
3. **纵向食材故事线**：7 个食材自上而下，每个配【左侧短标签 + 下方 2 行描述】
4. **底部主菜+飞溅动态**
5. **缩小徽章**（图标+1行短词）
6. **暖金光从顶部打下+暗背景**
