## 二、设计意图说明

### 为什么移除左侧所有冗余元素？
- v3 左侧堆叠了 6 个元素（印章×2 + 主标题 + 副标题 + 小字×3行），用户反馈"太杂"
- v4 只保留核心标题「常德牛肉米粉」，让视觉焦点回归**食材本身**和**金色标题**的质感
- 极简排版更接近**高定美食杂志**（如《Food & Wine》高端版）的留白美学

### 为什么统一深色背景？
- 移除红底色条后，三栏不再需要彩色区分——深色背景让金色标题 + 金色标签 + 金色虚线形成**统一的金属色系视觉线索**
- 深石板色比纯黑色更有层次，避免画面死黑

### 为什么换浅木色桌子？
- 原木色在深色背景中形成**温暖对比**，增加层次感
- 与土陶大碗、木筷形成自然的**材质呼应**（陶土 + 木）
- 浅木色桌面反射暖光，补充底部光线

### 标签框改为淡金色描边
- 移除红色后，用金色细框代替红色粗框，保持视觉统一
- 半透明深色底 + 金色细边 = 更高级的标签质感，类似高端餐厅菜单的信息卡片

## 五、Prompt 优化备忘

### 移除红色后的视觉补偿策略
| 移除项 | 补偿方式 |
|--------|----------|
| 红底色条 | 全图统一深石板背景，层次靠光影不靠色块 |
| 红色印章 | 用金色标题本身的金属反光和极淡暗金描边提供视觉锚点 |
| 红色标签框 | 淡金色细描边 + 半透明底，更高级 |
| 副标题/小字 | 靠标题字号更大（70%占比）和上下留白制造"少即是多"的高级感 |

### 桌子描述的关键词
`浅木色厚实原木桌面` / `日式桧木或橡木质感` / `天然木纹纹理` —— 这三个词确保 AI 理解是原木而非塑料或漆面。
