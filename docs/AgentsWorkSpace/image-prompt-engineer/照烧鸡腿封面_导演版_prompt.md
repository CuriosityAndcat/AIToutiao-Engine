### 🏮 暗调高级感（六选一，已锁定）

| 维度 | 设置 |
|------|------|
| **摄影机** | 静态俯拍锁定，无运动，让食材悬浮张力成为主角 |
| **灯光** | 电影级影棚逆光，主光从右后方 45° 打入，伦勃朗布光阴影深暗 |
| **色彩** | 高对比低饱和，暗石板/墨棕/古铜/琥珀金为主色调 |
| **节奏** | 极缓（静态图），长凝视食材质感 |
| **声音** | 静态图无音频，但视觉上呈现"酱汁黏稠下坠 + 蒸汽升腾"的张力 |
| **代表场景对应** | 对齐 cover1.png（牛肝菌焖饭）的暗调高级感调性 |

**为什么是暗调高级感而不是日杂温暖派？**
- 任务要求对齐 cover1.png 的视觉调性 — 而 cover1.png 正是暗调（深石板背景 + 金色点缀 + 蒸汽体积光）
- 照烧鸡腿的"焦糖化琥珀色"在暗背景下比在明亮日杂背景下更"发光"
- 信息图式分层悬浮结构（Exploded View）天然属于暗调高级感派系

## 三、视频弧线（静态图视觉节奏）

虽然这是静态图，但视觉上仍有"信息流"：

```
弧线: [open=顶层照烧汁倾泻] → [build=鸡腿排+溏心蛋逐层下沉]
     → [climax=碗中成品组装完成] → [landing=热气升腾+酱汁流动]

高潮点: 画面下方 1/3 的白色陶瓷碗（成品组装完成的瞬间）
        — 这是观众眼睛的最终落点，所有食材"汇聚于此"

景别趋势: 远景观察全图 → 视线垂直下移 → 特写锁定碗口蒸汽升腾
运动趋势: 静态构图 → 视线垂直下移（自上而下）→ 视线锁定碗口
光比趋势: 顶光打透蒸汽形成体积光 → 侧逆光勾出鸡腿排焦糖化表皮
         → 碗口暖光聚焦成品
色彩趋势: 顶部琥珀金（照烧汁）→ 中部深棕+金黄（鸡腿+蛋）
         → 下部翠绿+米白（菜+米）→ 碗口蒸汽白
```

**节奏型**：垂直向下的视觉引导（眼睛从「照烧汁」→ 鸡腿 → 蛋 → 菜 → 米饭 → 碗），类似电影开场镜头从天空坠入大地。

## 四、7 槽 Director Formula 编译

按 `[主体] + [动作] + [场景] + [运镜] + [光影/风格] + [音频] + [约束]` 优先级排列：

| 槽位 | 填充 |
|------|------|
| **主体** | 照烧鸡腿杂蔬饭的 5 层 Exploded View 解构（金色照烧汁→鸡腿排→溏心蛋→西兰花→珍珠米） |
| **动作** | 食材垂直悬浮分层，酱汁从顶向下"倾泻"感，碗中热气向上升腾 |
| **场景** | 深石板色背景 + 浅木色桌面 + 白色日式陶瓷深碗（碗口有热气） |
| **运镜** | 静态俯拍锁定（无运动，让食材悬浮张力说话） |
| **光影/风格** | 暗调高级感：电影级影棚逆光 + 蒸汽丁达尔体积光 + 焦糖化金色反光 |
| **音频** | 静态图无音频，但视觉上呈现"酱汁黏稠下坠 + 蒸汽升腾"的张力 |
| **约束** | 严格 5 层解构、每层只一种核心食材、禁混层、对齐 cover1.png 风格 |

# v2 更新记录（在 v1 基础上追加文字信息图层）

## 一、v2 设计意图

**问题诊断**：v1 的 5 层食材 + 暗调高级感 + 摄影质感**已经达成封面 90% 的视觉冲击力**，但作为小红书/抖音的美食封面，缺少「信息图核心价值」—— 观众第一眼无法 get 菜品名称、食材组成、烹饪亮点。

**v2 解决方案**：保留 v1 的全部摄影底图（5 层悬浮 + 底部成品 + 暗调逆光 + 蒸汽飞溅），**仅追加一层杂志级中文排版图层**，对齐 cover1.png（牛肝菌焖饭）的视觉范式。

**v2 核心设计原则**：
1. **双层结构**：摄影底图（5 层食材+碗） + 平面设计图层（文字+标签+虚线）= 杂志信息图
2. **空间分区**：左侧 1/6 标题区 / 中部 1/2 食材区 / 右侧 1/5 标签区（与 cover1.png 同款三栏比例）
3. **景深隔离**：文字图层必须在 85mm 镜头景深之外，绝对不能虚化
4. **不破坏食欲意图**：标题区在左、标签区在右、食材在中，三者形成「文字 ↔ 食材」的视觉对话，不挤压主视觉

| 编号 | 标签 | 描述 | 对应食材层 |

**标签视觉规格**：
- 白色/极浅米白色半透明矩形，圆角
- 极细浅金/浅灰边框（1px）
- 标签内左上角浅金色小圆圈编号 ①~⑤
- 食材名 = 黑色或深棕色粗体
- 描述 = 2 行白色或极浅灰小字

### 2.3 金色虚线连接

- 从每个标签左侧边缘中部水平伸出
- 线宽 1-2px，颜色 #C9A961 左右（浅金）
- 短横线-短横线-短横线虚线样式
- 水平向左横穿中部深色区域
- 精确终止于对应食材层右侧边缘
- 5 条虚线垂直间距与 5 层食材层层间距严格对齐

### 2.4 景深处理

> 文字图层必须在 85mm 微距镜头的景深之外（焦平面之外），绝对不能虚化或扭曲。
> 中文文字边缘锐利，印刷级清晰度。

**为什么强调景深之外？** 因为 85mm 微距镜头的浅景深会自然虚化焦平面外元素。如果不强调「文字必须在景深外但绝对清晰」，模型很可能把中文也虚化掉，文字会糊成一团。
