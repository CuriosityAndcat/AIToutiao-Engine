## 一、三项优化需求总览

| # | 需求 | 影响范围 | 优先级 |
|---|------|---------|--------|
| 1 | 所有文字统一中文简体 + 宋体 + 解决文字缺失部位（笔画不完整） | 【文字图层在画面中的处理】段 + 所有涉及字体的描述 | 🔴 高 |
| 2 | 去掉最上方光源，只保留右后方 45° 主光 | 【背景与桌面】段 + 【摄影与质感】段 | 🔴 高 |
| 3 | 桌面更大、更古朴（老榆木 + 量化占高 + 年轮/磨损/树皮边缘） | 参数表 `{桌面}` + 模板【背景与桌面】段 | 🟡 中 |
